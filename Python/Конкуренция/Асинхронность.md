# Асинхронность (Asynchrony / Async / Await)

**Асинхронность** — способ выполнения задач с кооперативным переключением при ожидании I/O. Один поток, одно ядро, тысячи задач.

## 1. Основные понятия

| Термин | Описание |
|---|---|
| **async** | Ключевое слово для объявления корутины |
| **await** | Ключевое слово для ожидания корутины |
| **coroutine** | Асинхронная функция |
| **event loop** | Цикл событий — диспетчер задач |
| **task** | Задача, обёртка над корутиной |
| **future** | Объект с результатом, который будет |

## 2. Базовый синтаксис

```python
import asyncio

async def greet(name):
    print(f"Привет, {name}!")
    await asyncio.sleep(1)  # неблокирующая задержка
    print(f"Пока, {name}!")

async def main():
    await greet("Анна")

asyncio.run(main())
```

## 3. Запуск нескольких задач

### gather

```python
async def fetch(url):
    await asyncio.sleep(1)
    return f"Data from {url}"

async def main():
    tasks = [fetch(f"url{i}") for i in range(3)]
    results = await asyncio.gather(*tasks)
    print(results)

asyncio.run(main())
```

### Task.create_task

```python
async def main():
    task1 = asyncio.create_task(fetch("url1"))
    task2 = asyncio.create_task(fetch("url2"))

    result1 = await task1
    result2 = await task2
```

### as_completed

```python
async def main():
    tasks = [fetch(f"url{i}") for i in range(5)]

    for coro in asyncio.as_completed(tasks):
        result = await coro
        print(f"Готово: {result}")
```

### wait

```python
async def main():
    tasks = [fetch(f"url{i}") for i in range(5)]
    done, pending = await asyncio.wait(tasks, timeout=2, return_when=FIRST_COMPLETED)

    for task in done:
        print(task.result())
```

## 4. Event Loop

```python
import asyncio

# получить event loop
loop = asyncio.get_event_loop()

# запустить корутину
loop.run_until_complete(main())

# закрыть
loop.close()

# asyncio.run() делает то же самое
asyncio.run(main())
```

## 5. await awaitable

`await` работает с тремя типами объектов:

```python
# 1. coroutine
async def coro():
    return 42

result = await coro()

# 2. Task (уже запущенная корутина)
task = asyncio.create_task(coro())
result = await task

# 3. Future
future = asyncio.Future()
future.set_result(42)
result = await future
```

## 6. sleep и timeout

```python
import asyncio

# sleep — неблокирующая задержка
await asyncio.sleep(1)

# timeout через asyncio.wait_for
try:
    result = await asyncio.wait_for(coro(), timeout=5)
except asyncio.TimeoutError:
    print("Таймаут")

# timeout через asyncio.timeout (3.11+)
try:
    async with asyncio.timeout(5):
        result = await coro()
except TimeoutError:
    print("Таймаут")
```

## 7. Shield — защита от отмены

```python
task = asyncio.create_task(coro())

# задача не будет отменена при отмене внешней
await asyncio.shield(task)

# принудительная отмена
task.cancel()
try:
    await task
except asyncio.CancelledError:
    print("Задача отменена")
```

## 8. Queue

```python
import asyncio

async def producer(queue):
    for i in range(5):
        await queue.put(i)
        print(f"Произвёл: {i}")
        await asyncio.sleep(0.5)

async def consumer(queue):
    while True:
        item = await queue.get()
        if item is None:
            break
        print(f"Потребил: {item}")
        queue.task_done()

async def main():
    queue = asyncio.Queue(maxsize=3)
    await asyncio.gather(
        producer(queue),
        consumer(queue),
    )
    await queue.join()

asyncio.run(main())
```

## 9. Semaphore (ограничение конкурентности)

```python
import asyncio

sem = asyncio.Semaphore(3)

async def bounded_task(url):
    async with sem:
        print(f"Загрузка {url}")
        await asyncio.sleep(1)
        return url

async def main():
    tasks = [bounded_task(f"url{i}") for i in range(10)]
    results = await asyncio.gather(*tasks)
    print(results)

asyncio.run(main())
```

## 10. Subprocess

```python
import asyncio

async def run_command(cmd):
    proc = await asyncio.create_subprocess_shell(
        cmd,
        stdout=asyncio.subprocess.PIPE,
        stderr=asyncio.subprocess.PIPE,
    )
    stdout, stderr = await proc.communicate()
    return stdout.decode()

async def main():
    result = await run_command("echo Hello")
    print(result)

asyncio.run(main())
```

## 12. Практические примеры

### HTTP-запросы

```python
import aiohttp
import asyncio

async def fetch(session, url):
    async with session.get(url) as response:
        return await response.text()

async def main():
    urls = ["https://example.com"] * 10
    async with aiohttp.ClientSession() as session:
        tasks = [fetch(session, url) for url in urls]
        results = await asyncio.gather(*tasks)

asyncio.run(main())
```

### Таймер

```python
async def tick():
    for i in range(10):
        print(f"Tick {i}")
        await asyncio.sleep(1)

async def tock():
    for i in range(10):
        print(f"Tock {i}")
        await asyncio.sleep(1.5)

async def main():
    await asyncio.gather(tick(), tock())

asyncio.run(main())
```

### Producer-Consumer

```python
import asyncio
import random

async def producer(queue):
    for i in range(10):
        await asyncio.sleep(random.uniform(0.1, 0.5))
        item = f"item-{i}"
        await queue.put(item)
        print(f"→ {item}")
    await queue.put(None)

async def consumer(name, queue):
    while True:
        item = await queue.get()
        if item is None:
            queue.task_done()
            break
        print(f"  {name} ← {item}")
        await asyncio.sleep(random.uniform(0.2, 0.4))
        queue.task_done()

async def main():
    queue = asyncio.Queue()
    await asyncio.gather(
        producer(queue),
        consumer("A", queue),
        consumer("B", queue),
    )

asyncio.run(main())
```

## 13. sync vs async

```python
# синхронно — блокирующий I/O
import requests

def fetch(url):
    response = requests.get(url)
    return response.text

# асинхронно — неблокирующий I/O
import aiohttp

async def fetch(url):
    async with aiohttp.ClientSession() as session:
        async with session.get(url) as response:
            return await response.text()
```

| | sync | async |
|---|---|---|
| Потоков | 1 задача | Много задач |
| I/O | Блокируется | Не блокируется |
| Параллелизм | threading/process | asyncio |
| Сложность | Ниже | Выше |
| Производительность I/O | Ниже | Выше |

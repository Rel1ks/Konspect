# Инкапсуляция (Encapsulation)

**Инкапсуляция** — сокрытие внутреннего состояния объекта и предоставление доступа только через публичные методы.

## 1. Уровни доступа

В Python нет строгих модификаторов доступа, есть соглашения:

| Уровень | Синтаксис | Описание |
|---|---|---|
| Публичный | `self.name` | Доступен отовсюду |
| Защищённый | `self._name` | Только внутри класса и потомков (по соглашению) |
| Приватный | `self.__name` | Name mangling — `_ClassName__name` |

```python
class Person:
    def __init__(self, name, age, pin):
        self.name = name        # публичный
        self._age = age         # защищённый
        self.__pin = pin        # приватный

    def info(self):
        return f"{self.name}, {self._age}"

p = Person("Анна", 25, "1234")
print(p.name)         # Анна — ок
print(p._age)         # 25 — работает, но "не рекомендуется"
# print(p.__pin)      # AttributeError — name mangling
print(p._Person__pin) # 1234 — доступ через mangling
```

## 2. Name mangling (искажение имени)

Python автоматически переименовывает `__name` в `_ClassName__name`.

```python
class A:
    def __init__(self):
        self.__secret = "секрет"

    def get_secret(self):
        return self.__secret

class B(A):
    def __init__(self):
        super().__init__()
        self.__secret = "другой секрет"

a = A()
b = B()

# у каждого своя версия __secret из-за name mangling
print(a._A__secret)   # секрет
print(b._A__secret)   # секрет (родительский)
print(b._B__secret)   # другой секрет (дочерний)
```

### Предназначение name mangling

```python
class Base:
    def __init__(self):
        self.__value = 10

    def get_value(self):
        return self.__value

class Derived(Base):
    def __init__(self):
        super().__init__()
        self.__value = 99  # не перезаписывает Base.__value

d = Derived()
print(d.get_value())       # 10 (родительский __value)
print(d._Derived__value)   # 99 (свой __value)
```

## 3. Геттеры и сеттеры (старый стиль)

```python
class Person:
    def __init__(self, name, age):
        self._name = name
        self._age = age

    # геттер
    def get_age(self):
        return self._age

    # сеттер
    def set_age(self, value):
        if value < 0:
            raise ValueError("Возраст не может быть отрицательным")
        if value > 150:
            raise ValueError("Некорректный возраст")
        self._age = value

    # удаление
    def delete_age(self):
        print("Возраст удалён")
        del self._age

p = Person("Анна", 25)
print(p.get_age())
p.set_age(30)
# p.set_age(-5)  # ValueError
```

## 4. `@property` (современный стиль)

```python
class Person:
    def __init__(self, name, age):
        self._name = name
        self._age = age

    @property
    def age(self):
        """Геттер — вызывается при чтении"""
        return self._age

    @age.setter
    def age(self, value):
        """Сеттер — вызывается при присваивании"""
        if value < 0:
            raise ValueError("Возраст не может быть отрицательным")
        if value > 150:
            raise ValueError("Некорректный возраст")
        self._age = value

    @age.deleter
    def age(self):
        """Deleter — вызывается при del"""
        print("Возраст удалён")
        del self._age

    @property
    def is_adult(self):
        """Вычисляемое свойство (только чтение)"""
        return self._age >= 18

p = Person("Анна", 25)
print(p.age)         # 25 (геттер)
p.age = 30           # сеттер
# p.age = -5         # ValueError
print(p.is_adult)    # True (вычисляемое свойство)
del p.age            # deleter
```

## 5. Property через дескриптор

```python
class PositiveNumber:
    def __get__(self, obj, objtype=None):
        if obj is None:
            return self
        return obj.__dict__.get(self.name, 0)

    def __set__(self, obj, value):
        if value < 0:
            raise ValueError(f"{self.name} должен быть положительным")
        obj.__dict__[self.name] = value

    def __set_name__(self, owner, name):
        self.name = f"_{name}"

class Order:
    price = PositiveNumber()
    quantity = PositiveNumber()

    def __init__(self, price, quantity):
        self.price = price
        self.quantity = quantity

    def total(self):
        return self.price * self.quantity

order = Order(100, 2)
print(order.total())    # 200
# order.price = -50     # ValueError
```

## 6. Property в наследниках

```python
class Animal:
    def __init__(self, name):
        self._name = name

    @property
    def name(self):
        return self._name.upper()

    @name.setter
    def name(self, value):
        self._name = value

class Dog(Animal):
    @property
    def name(self):
        return f"Пёс {self._name}"

    @name.setter
    def name(self, value):
        self._name = f"{value} (собака)"

dog = Dog("Бобик")
print(dog.name)       # Пёс Бобик
dog.name = "Рекс"
print(dog.name)       # Пёст Рекс (собака)
```

## 7. Скрытие данных через `__slots__`

`__slots__` ограничивает атрибуты, которые можно создать.

```python
class Person:
    __slots__ = ("name", "age")

    def __init__(self, name, age):
        self.name = name
        self.age = age

    def info(self):
        return f"{self.name}, {self.age}"

p = Person("Анна", 25)
print(p.info())

# p.city = "Москва"  # AttributeError — нет slots для city
# __slots__ экономит память и предотвращает случайные атрибуты
```

## 8. Свойства только для чтения

```python
class Config:
    def __init__(self):
        self._values = {
            "version": "1.0",
            "debug": True,
        }

    @property
    def version(self):
        return self._values["version"]

    # только getter — setter не определён
    # config.version = "2.0"  # AttributeError

config = Config()
print(config.version)
# config.version = "2.0"  # ошибка
```

## 9. Вычисляемые свойства

```python
class Rectangle:
    def __init__(self, width, height):
        self._width = width
        self._height = height

    @property
    def width(self):
        return self._width

    @width.setter
    def width(self, value):
        if value <= 0:
            raise ValueError("Ширина должна быть положительной")
        self._width = value

    @property
    def height(self):
        return self._height

    @height.setter
    def height(self, value):
        if value <= 0:
            raise ValueError("Высота должна быть положительной")
        self._height = value

    @property
    def area(self):
        return self._width * self._height

    @property
    def perimeter(self):
        return 2 * (self._width + self._height)

r = Rectangle(10, 5)
print(r.area)        # 50
print(r.perimeter)   # 30
r.width = 20
print(r.area)        # 100
# r.area = 200       # AttributeError — свойство только для чтения
```

## 10. `@cached_property`

Кеширует результат вычисления до следующего изменения.

```python
from functools import cached_property
import time

class DataLoader:
    def __init__(self, url):
        self.url = url
        self._data = None

    @cached_property
    def data(self):
        """Вычисляется один раз, потом кешируется"""
        print(f"Загрузка данных с {self.url}...")
        time.sleep(2)  # имитация загрузки
        return f"данные с {self.url}"

loader = DataLoader("https://api.example.com")
print(loader.data)   # загрузка (2 сек)
print(loader.data)   # из кеша (мгновенно)
```

## 11. Сравнение подходов

| Подход | Гибкость | Шум | Использование |
|---|---|---|---|
| Публичный атрибут | Высокая | Низкий | Простые данные |
| `_защищённый` | Высокая | Средний | Внутреннее использование |
| `__приватный` | Низкая (mangling) | Высокий | Избегать коллизий в наследниках |
| Геттер/сеттер | Высокая | Высокий | Старый стиль |
| `@property` | Высокая | Средний | Современный стиль |

### Рекомендации

```python
# 1. Обычный атрибут — для простых случаев
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

# 2. @property — когда нужна валидация или вычисления
class Temperature:
    def __init__(self, celsius):
        self._celsius = celsius

    @property
    def celsius(self):
        return self._celsius

    @celsius.setter
    def celsius(self, value):
        if value < -273.15:
            raise ValueError("Ниже абсолютного нуля")
        self._celsius = value

    @property
    def fahrenheit(self):
        return self._celsius * 9 / 5 + 32

# 3. _защищённый — для внутренних деталей
class Database:
    def __init__(self):
        self._connection = None

    def connect(self):
        self._connection = "connected"

# 4. __приватный — только если нужен name mangling
class Base:
    def __init__(self):
        self.__value = 10  # избежать переопределения в потомках
```

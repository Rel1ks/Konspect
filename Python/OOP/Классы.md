# Классы (Classes) в Python

**Класс** — шаблон для создания объектов. Объединяет данные (атрибуты) и методы (функции) в одну структуру.

## 1. Определение класса

```python
class Person:
    pass  # пустой класс

# создание объекта (экземпляра)
p = Person()
print(type(p))  # <class '__main__.Person'>
```

## 2. Метод `__init__` (конструктор)

Вызывается автоматически при создании объекта.

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

p = Person("Анна", 25)
print(p.name)  # Анна
print(p.age)   # 25
```

## 3. Атрибуты

### Атрибуты экземпляра

Принадлежат конкретному объекту.

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

p1 = Person("Анна", 25)
p2 = Person("Борис", 30)

print(p1.name)  # Анна
print(p2.name)  # Борис
```

### Атрибуты класса

Принадлежат самому классу, общие для всех экземпляров.

```python
class Person:
    species = "человек"  # атрибут класса

    def __init__(self, name):
        self.name = name  # атрибут экземпляра

print(Person.species)  # человек
p = Person("Анна")
print(p.species)       # человек
```

## 4. Методы

### Обычные методы (instance methods)

Первый параметр — `self` (ссылка на экземпляр).

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"Привет, я {self.name}!"

    def is_adult(self):
        return self.age >= 18

p = Person("Анна", 25)
print(p.greet())
print(p.is_adult())
```

### Методы класса (@classmethod)

Первый параметр — `cls` (ссылка на класс).

```python
class Person:
    count = 0

    def __init__(self, name):
        self.name = name
        Person.count += 1

    @classmethod
    def total_people(cls):
        return f"Всего людей: {cls.count}"

    @classmethod
    def from_birth_year(cls, name, year):
        age = 2026 - year
        return cls(name, age)

p1 = Person("Анна")
p2 = Person("Борис")
print(Person.total_people())  # Всего людей: 2

p3 = Person.from_birth_year("Вика", 2000)
print(p3.age)
```

### Статические методы (@staticmethod)

Не принимают ни `self`, ни `cls`. Обычные функции внутри класса.

```python
class Math:
    @staticmethod
    def add(a, b):
        return a + b

    @staticmethod
    def multiply(a, b):
        return a * b

print(Math.add(5, 3))
print(Math.multiply(5, 3))
```

## 5. `self`

`self` — ссылка на текущий экземпляр. Можно называть по-другому, но принято `self`.

```python
class Person:
    def __init__(self, name):
        self.name = name

    def rename(self, new_name):
        self.name = new_name

p = Person("Анна")
p.rename("Вика")
print(p.name)  # Вика

# это эквивалентно:
Person.rename(p, "Вика")
```

## 6. Специальные (магические) методы

| Метод | Описание | Пример |
|---|---|---|
| `__init__` | Конструктор | `obj = Class(args)` |
| `__str__` | Строковое представление (для пользователя) | `str(obj)`, `print(obj)` |
| `__repr__` | Строковое представление (для разработчика) | `repr(obj)` |
| `__len__` | Длина | `len(obj)` |
| `__eq__` | Сравнение == | `obj1 == obj2` |
| `__lt__` | Сравнение < | `obj1 < obj2` |
| `__add__` | Сложение + | `obj1 + obj2` |
| `__getitem__` | Доступ по индексу | `obj[key]` |
| `__setitem__` | Установка по индексу | `obj[key] = val` |
| `__call__` | Вызов объекта как функции | `obj(args)` |
| `__bool__` | Преобразование в bool | `if obj:` |

```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __str__(self):
        return f"Vector({self.x}, {self.y})"

    def __repr__(self):
        return f"Vector({self.x}, {self.y})"

    def __add__(self, other):
        return Vector(self.x + other.x, self.y + other.y)

    def __eq__(self, other):
        return self.x == other.x and self.y == other.y

    def __len__(self):
        return int((self.x ** 2 + self.y ** 2) ** 0.5)

    def __bool__(self):
        return self.x != 0 or self.y != 0

    def __call__(self):
        return f"Вызов Vector({self.x}, {self.y})"

v1 = Vector(3, 4)
v2 = Vector(1, 2)

print(v1)              # Vector(3, 4)
print(v1 + v2)         # Vector(4, 6)
print(v1 == Vector(3, 4))  # True
print(len(v1))         # 5
print(bool(Vector(0, 0)))  # False
print(v1())            # Вызов Vector(3, 4)
```

## 7. Инкапсуляция

В Python нет строгой инкапсуляции, есть соглашения:

| Соглашение | Пример | Значение |
|---|---|---|
| Публичный | `self.name` | Доступен отовсюду |
| Защищённый | `self._name` | Внутреннее использование (по соглашению) |
| Приватный | `self.__name` | Name mangling — `_ClassName__name` |

```python
class BankAccount:
    def __init__(self, owner, balance):
        self.owner = owner      # публичный
        self._balance = balance  # защищённый
        self.__pin = "1234"      # приватный

    def get_balance(self):
        return self._balance

    def deposit(self, amount):
        if amount > 0:
            self._balance += amount

    def _validate(self):
        pass  # внутренний метод

acc = BankAccount("Анна", 1000)
print(acc.owner)
print(acc._balance)        # доступно, но "не рекомендуется"
print(acc._BankAccount__pin)  # name mangling
```

## 8. Свойства (@property)

Позволяют управлять доступом к атрибутам через геттеры/сеттеры.

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self._age = age

    @property
    def age(self):
        return self._age

    @age.setter
    def age(self, value):
        if value < 0:
            raise ValueError("Возраст не может быть отрицательным")
        self._age = value

    @age.deleter
    def age(self):
        print("Удаление возраста")
        del self._age

    @property
    def is_adult(self):
        return self._age >= 18

p = Person("Анна", 25)
print(p.age)       # 25
p.age = 30
# p.age = -5       # ValueError
print(p.is_adult)  # True (вызывается как атрибут)
```

## 9. Вложенные классы

```python
class Car:
    def __init__(self, brand):
        self.brand = brand
        self.engine = self.Engine()

    class Engine:
        def __init__(self):
            self.horsepower = 150

        def start(self):
            return "Двигатель запущен"

car = Car("Toyota")
print(car.engine.start())
print(car.engine.horsepower)
```

## 10. Класс как декоратор

```python
class CountCalls:
    def __init__(self, func):
        self.func = func
        self.count = 0

    def __call__(self, *args, **kwargs):
        self.count += 1
        print(f"Вызов #{self.count}")
        return self.func(*args, **kwargs)

@CountCalls
def say_hello():
    print("Привет!")

say_hello()
say_hello()
```

## 11. Сравнение классов

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def __eq__(self, other):
        if not isinstance(other, Person):
            return NotImplemented
        return self.name == other.name and self.age == other.age

    def __lt__(self, other):
        return self.age < other.age

    def __hash__(self):
        return hash((self.name, self.age))

p1 = Person("Анна", 25)
p2 = Person("Анна", 25)
p3 = Person("Борис", 30)

print(p1 == p2)  # True
print(p1 < p3)   # True

# можно использовать в set и dict
people = {p1, p3}
print(people)
```

## 12. Именованные кортежи и dataclasses

```python
from dataclasses import dataclass

@dataclass
class Person:
    name: str
    age: int
    city: str = "Москва"

    def greet(self):
        return f"Привет, я {self.name}!"

p = Person("Анна", 25)
print(p)              # Person(name='Анна', age=25, city='Москва')
print(p.greet())

# __init__, __repr__, __eq__ уже сгенерированы
p2 = Person("Анна", 25)
print(p == p2)        # True
```

## Базовые команды

### Ввод данных

Функция: `input()`
В скобках к любой функции указывается какой-либо аргумент (скобки всегда идут с вызовом функций). Результатом выполнения данной функции служат возврат значения, введенного с клавиатуры, в память компьютера. #input

### Вывод данных

Функция: `print()`
В качестве аргумента функция `print()` может принимать числа, строки, математические выражения или ничего (пустая строка) #print

### Определение типа объекта

Функция: `type()`
Эта функция возвращает информацию о том, к какому классу принадлежит объект. #type

## Переменная

**Переменная** - именованная область памяти компьютера, в которой записаны какие-либо данные. Значение переменной называется **литералом** и может представлять несколько типов данных. #Переменная #Литерал

Чтобы объявить переменную в python, используется оператор присваивания `=`. В левой части имя переменной, в правой ее литерал.

Вид:
```python
 <Имя_переменной> = <Литерал>
```

## Типы данных

1. Целые числа (Integer) - **int**. Пример: 123 #int 
2. Вещественные числа (Float) - **float**. Пример: 12.3 #float 
3. Логический тип (Boolean) - **bool**. Пример: True, False #bool 
4. Строковый тип (String) - **str**. Пример: "Hello" #str

## 6. Асинхронность

| Пакет | Описание | Команда |
|---|---|---|
| **asyncio** | Встроенная асинхронность | Встроен |
| **aiohttp** | Асинхронный HTTP | `pip install aiohttp` |
| **httpx** | HTTP/2 + async | `pip install httpx` |
| **aiosqlite** | Асинхронная SQLite | `pip install aiosqlite` |

```python
import asyncio
import aiohttp

async def fetch(url):
    async with aiohttp.ClientSession() as session:
        async with session.get(url) as response:
            return await response.text()

async def main():
    content = await fetch("https://example.com")
    print(len(content))

asyncio.run(main())
```

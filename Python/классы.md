### Ведение 

Класс - это пользовательский тип данных, конструкции ООП (Объектное-Ориентированного Программирования), которая описывает тот или иной объект, его свойства и способы взаимодействия с ним.

Класс имеет поля - переменные которые будут принадлежать к его экземпляру а также методы - функции которые также можно будет использовать для его экземпляра.

Простыми словами, класс - шаблон для реализации какого-то объекта. Например, мы знаем как должно выглядеть кружка какие характеристики и действия она имеет При помощи класса мы описываем "абстрактную мама-кружку" а экземпляры будут являться "настоящими, конкретными" кружками с определёнными характеристиками и действиями "мамы - кружки".

Класс решает несколько важных задач:
- Дублирование кода
- Унификация объектов (т.е. одинаковые по задумке объекты действия будут иметь одинаковые характеристики и функционал)


### Классы в Python 

Объявляется при помощи ключевого слова class и указания имени класса. Затем ставится двоеточие и со следующей строки описывается тело класса. 

```python
class Python:
	pass
```

Чтобы  объявить поля класса используется ключевое слово self, после которого, через точку указывается имя поля. Поле тоже самое и переменная, но внутри класса. Однако, через self к переменной можно обращаться только из тела метода. В параметрах метода также обязательно должен быть указан self.

```python
class Python:
	def test(self):
		self.msg = "Hello!"
```

Для инициализации полей при создании экземпляра класса используется функция-конструктор. Она обязательно принимает параметр self.Эта функция будет автоматически вызвана при создании экземпляра класса. Если она не объявлена, будет использован конструктор по умолчанию (пустой).

```python
class Python:
	
	def __init__(self):
		self.msg = "Hello"
```

Создание экземпляра класса (объекта класса, объект) происходит вне тела класса, при помощи указания имени класса и круглых скобок после имени. Если в конструкции кроме self указаны другие параметры, то при  создании объекта в скобках мы также укажем атрибуты - то, что является параметрами при описании

```python
class Python:
	def __init__(self, text):
		self.msg = text
```

# Менеджеры пакетов Python

**Менеджер пакетов** — инструмент для установки, обновления, удаления и управления зависимостями пакетов.

## Зачем нужны менеджеры пакетов

- Установка сторонних библиотек
- Управление версиями зависимостей
- Изоляция окружений (чтобы проекты не конфликтовали)
- Воспроизводимость (фиксация версий)
- Публикация собственных пакетов

## Эволюция

```
pip (2008) → pip + virtualenv
              ↓
pip + venv (встроенный, Python 3.3+)
    ↓
Poetry (2018) — зависимости + сборка + публикация
PDM  (2020) — PEP 582/621
uv   (2023) — Rust, сверхбыстрый
Conda — не только Python, бинарные пакеты
```

## Сравнение

| Инструмент | Год | Lock-файл | Скорость | Особенность |
|---|---|---|---|---|
| **pip** | 2008 | `requirements.txt` | 🟡 Средняя | Стандарт, простота |
| **Poetry** | 2018 | `poetry.lock` | 🟡 Средняя | Всё в одном |
| **PDM** | 2020 | `pdm.lock` | 🟢 Высокая | PEP 582/621 |
| **uv** | 2023 | `uv.lock` | 🔵 Очень высокая | Rust, замена pip |
| **Conda** | 2012 | `environment.yml` | 🔴 Низкая | Не только Python |

## Что внутри

| Инструмент | Разрешение зависимостей | Вендоринг | Кеш | Parallel install |
|---|---|---|---|---|
| pip | Нет (линейная установка) | Нет | Да | Нет |
| Poetry | Да (SAT solver) | Нет | Да | Нет |
| PDM | Да (SAT solver) | Нет | Да | Да |
| uv | Да (SAT solver) | Да | Да | Да |
| Conda | Да (SAT solver) | Нет | Да | Да |

## Репозитории

| Репозиторий | Формат | Команда |
|---|---|---|
| **PyPI** (pypi.org) | sdist + wheel | `pip install` |
| **Conda-forge** | tar.bz2 / .conda | `conda install -c conda-forge` |
| **TestPyPI** (test.pypi.org) | sdist + wheel | `pip install -i https://test.pypi.org/simple/` |
| **Частный/зеркало** | sdist + wheel | `pip install -i <url>` |

## Когда что использовать

```bash
# простота — pip + venv
python -m venv .venv
.venv\Scripts\activate
pip install requests
pip freeze > requirements.txt

# современный подход — Poetry или PDM
poetry new my-project && poetry add requests
pdm init && pdm add requests

# скорость — uv
uv venv && uv pip install requests

# научные пакеты — conda
conda create -n my-env python=3.12 numpy pandas
```

## Основные понятия

| Термин | Описание |
|---|---|
| **wheel** | Готовый к установке бинарный формат (`.whl`) |
| **sdist** | Исходный код (`.tar.gz`) |
| **PyPI** | Python Package Index — официальный репозиторий |
| **Lock-файл** | Фиксирует точные версии всех зависимостей |
| **Editable install** | `pip install -e .` — ссылка на исходники |
| **Dev-dependencies** | Зависимости для разработки (pytest, ruff) |
| **SemVer** | `major.minor.patch` — семантическое версионирование |

# Рекурсия (Recursion)

**Рекурсия** — когда функция вызывает саму себя. Состоит из **базового случая** (условие выхода) и **рекурсивного случая** (вызов себя с изменёнными аргументами).

## 1. Структура рекурсии

```python
def recursive_func(params):
    # базовый случай — выход
    if условие_выхода:
        return результат

    # рекурсивный случай — вызов себя
    return recursive_func(изменённые_params)
```

```python
def countdown(n):
    if n <= 0:       # базовый случай
        print("Старт!")
        return
    print(n)          # действие
    countdown(n - 1)  # рекурсивный случай

countdown(5)
```

## 2. Стек вызовов

Каждый рекурсивный вызов добавляется в стек. При достижении базового случая стек разворачивается.

```python
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)

# factorial(4):
#   4 * factorial(3)
#       3 * factorial(2)
#           2 * factorial(1)
#               1
#           2 * 1 = 2
#       3 * 2 = 6
#   4 * 6 = 24
```

```
Стек вызовов:
factorial(1) → 1
factorial(2) → 2 * 1
factorial(3) → 3 * 2
factorial(4) → 4 * 6
```

### Палиндром

```python
def is_palindrome(s):
    if len(s) <= 1:
        return True
    if s[0] != s[-1]:

    return is_palindrome(s[1:-1])
```

## 4. Рекурсия против цикла

Любую рекурсию можно заменить циклом (и наоборот).

```python
# рекурсия
def factorial_rec(n):
    if n <= 1:
        return 1
    return n * factorial_rec(n - 1)

# цикл
def factorial_loop(n):
    result = 1
    for i in range(2, n + 1):
        result *= i
    return result

# когда рекурсия естественнее: деревья, графы, комбинаторика
# когда цикл лучше: простые итерации, большая глубина
```

### Хвостовая рекурсия

Хвостовая рекурсия — когда рекурсивный вызов последний. Python **не оптимизирует** хвостовую рекурсию.

```python
# Хвостовая рекурсия (в Python не оптимизируется)
def factorial_tail(n, acc=1):
    if n <= 1:
        return acc
    return factorial_tail(n - 1, acc * n)

# Лучше использовать цикл
```

## 6. Мемоизация (кеширование)

Оптимизация рекурсии — сохранять результаты подзадач.

```python
from functools import lru_cache

# вариант 1: декоратор
@lru_cache(maxsize=None)
def fib(n):
    if n <= 1:
        return n
    return fib(n - 1) + fib(n - 2)

print(fib(50))  # 12586269025 — мгновенно

# вариант 2: вручную
memo = {}
def fib_manual(n):
    if n in memo:
        return memo[n]
    if n <= 1:
        return n
    memo[n] = fib_manual(n - 1) + fib_manual(n - 2)
    return memo[n]

fib_manual.cache_info = lambda: None
print(fib_manual(50))
```

```python
def dfs(graph, node, visited=None):
    if visited is None:
        visited = set()
    visited.add(node)
    print(node, end=" ")
    for neighbor in graph[node]:
        if neighbor not in visited:
            dfs(graph, neighbor, visited)
```

```python
def hanoi(n, source, target, auxiliary):
    if n == 1:
        print(f"Диск 1: {source} → {target}")
        return
    hanoi(n - 1, source, auxiliary, target)
```

### Комбинации и перестановки

```python
def combinations(arr, k):
    """Все сочетания по k элементов"""
    result = []

    def backtrack(start, current):
        if len(current) == k:
            result.append(current[:])
            return
        for i in range(start, len(arr)):
            current.append(arr[i])
            backtrack(i + 1, current)
            current.pop()

    backtrack(0, [])
    return result

def permutations(arr):
    """Все перестановки"""
    result = []

    def backtrack(current, remaining):
        if not remaining:
            result.append(current[:])
            return
        for i in range(len(remaining)):
            current.append(remaining[i])
            backtrack(current, remaining[:i] + remaining[i+1:])
            current.pop()

    backtrack([], arr)
    return result

print(combinations([1, 2, 3, 4], 2))
print(permutations([1, 2, 3]))
```

### Подсчёт путей в сетке

```python
def count_paths(m, n, memo=None):
    """Количество путей из (0,0) в (m-1, n-1) — только вправо и вниз"""
    if memo is None:
        memo = {}
    if (m, n) in memo:
        return memo[(m, n)]
    if m == 1 or n == 1:
        return 1
    memo[(m, n)] = count_paths(m - 1, n, memo) + count_paths(m, n - 1, memo)
    return memo[(m, n)]

print(count_paths(3, 3))  # 6
```

## 8. Разделяй и властвуй (Divide and Conquer)

```python
def binary_search(arr, target, low=0, high=None):
    if high is None:
        high = len(arr) - 1
    if low > high:
        return -1
    mid = (low + high) // 2
    if arr[mid] == target:
        return mid
    if target < arr[mid]:
        return binary_search(arr, target, low, mid - 1)
    return binary_search(arr, target, mid + 1, high)

arr = [1, 3, 5, 7, 9, 11]
print(binary_search(arr, 7))   # 3
print(binary_search(arr, 4))   # -1
```

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr

    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])

    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result

print(merge_sort([3, 1, 4, 1, 5, 9, 2, 6]))
```

## 9. Backtracking (поиск с возвратом)

```python
def solve_n_queens(n):
    """Расставить n ферзей на доске n×n"""
    cols = set()
    diag1 = set()  # r + c
    diag2 = set()  # r - c
    result = []

    def backtrack(row, board):
        if row == n:
            result.append(board[:])
            return
        for col in range(n):
            if col in cols or (row + col) in diag1 or (row - col) in diag2:
                continue
            cols.add(col)
            diag1.add(row + col)
            diag2.add(row - col)
            board.append(col)

            backtrack(row + 1, board)

            board.pop()
            cols.remove(col)
            diag1.remove(row + col)
            diag2.remove(row - col)

    backtrack(0, [])
    return result

solutions = solve_n_queens(4)
for sol in solutions:
    print(sol)
```

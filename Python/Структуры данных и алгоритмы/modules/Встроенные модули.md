# Встроенные модули (Built-in)

**Встроенные модули** — это модули, которые поставляются вместе с Python. Их не нужно устанавливать — только импортировать.

## 1. `math` — математические функции

| Функция / константа | Описание | Пример |
|---|---|---|
| `math.pi` | Число π | `3.141592653589793` |
| `math.e` | Число e | `2.718281828459045` |
| `math.sqrt(x)` | Квадратный корень | `sqrt(16)` → 4.0 |
| `math.pow(x, y)` | Возведение в степень | `pow(2, 3)` → 8.0 |
| `math.floor(x)` | Округление вниз | `floor(3.9)` → 3 |
| `math.ceil(x)` | Округление вверх | `ceil(3.1)` → 4 |
| `math.trunc(x)` | Отбрасывание дробной части | `trunc(3.9)` → 3 |
| `math.factorial(x)` | Факториал | `factorial(5)` → 120 |
| `math.gcd(a, b)` | НОД | `gcd(12, 18)` → 6 |
| `math.fabs(x)` | Модуль числа (float) | `fabs(-5)` → 5.0 |
| `math.sin(x)`, `cos(x)`, `tan(x)` | Тригонометрия | `sin(pi/2)` |
| `math.degrees(x)` | Радианы → градусы | `degrees(pi)` → 180.0 |
| `math.radians(x)` | Градусы → радианы | `radians(180)` → π |

```python
import math
print(math.sqrt(144))
print(math.factorial(6))
print(math.gcd(24, 36))
```

## 2. `random` — случайные числа

| Функция | Описание | Пример |
|---|---|---|
| `random.random()` | Случайное float от 0 до 1 | `0.374...` |
| `random.randint(a, b)` | Случайное int в [a, b] | `randint(1, 6)` |
| `random.uniform(a, b)` | Случайное float в [a, b] | `uniform(0, 10)` |
| `random.choice(seq)` | Случайный элемент | `choice(["a", "b", "c"])` |
| `random.shuffle(lst)` | Перемешать список | `shuffle(cards)` |
| `random.sample(pop, k)` | Выборка без повторений | `sample(range(10), 3)` |
| `random.seed(n)` | Фиксация генератора | `seed(42)` |

```python
import random
print(random.randint(1, 10))
print(random.choice(["орёл", "решка"]))
items = [1, 2, 3, 4, 5]
random.shuffle(items)
print(items)
```

## 3. `datetime` — дата и время

| Тип / функция | Описание | Пример |
|---|---|---|
| `datetime.date` | Дата (год, месяц, день) | `date(2024, 1, 15)` |
| `datetime.time` | Время (час, минута, секунда) | `time(14, 30)` |
| `datetime.datetime` | Дата + время | `datetime(2024, 1, 15, 14, 30)` |
| `datetime.now()` | Текущая дата/время | `datetime.now()` |
| `date.today()` | Сегодняшняя дата | `date.today()` |
| `timedelta` | Разница во времени | `timedelta(days=7)` |

```python
from datetime import datetime, date, timedelta

now = datetime.now()
print(now)
print(now.strftime("%d.%m.%Y %H:%M"))

today = date.today()
week_later = today + timedelta(days=7)
print(week_later)
```

## 6. `json` — работа с JSON

| Функция | Описание | Пример |
|---|---|---|
| `json.dumps(obj)` | Python → JSON строка | `dumps({"a": 1})` |
| `json.loads(s)` | JSON строка → Python | `loads('{"a": 1}')` |
| `json.dump(obj, file)` | Python → JSON файл | `dump(data, f)` |
| `json.load(file)` | JSON файл → Python | `load(f)` |

```python
import json

data = {"name": "Анна", "age": 25, "is_student": False}
json_str = json.dumps(data, ensure_ascii=False, indent=2)
print(json_str)

parsed = json.loads(json_str)
print(parsed["name"])

# запись в файл
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

# чтение из файла
with open("data.json", "r", encoding="utf-8") as f:
    loaded = json.load(f)
```

## 7. `re` — регулярные выражения

| Функция | Описание | Пример |
|---|---|---|
| `re.search(pattern, s)` | Поиск подстроки | `search(r"\d+", "abc123")` |
| `re.match(pattern, s)` | Совпадение с начала | `match(r"\d+", "123abc")` |
| `re.findall(pattern, s)` | Все совпадения | `findall(r"\d+", "a1b2c3")` |
| `re.sub(pattern, repl, s)` | Замена | `sub(r"\d+", "X", "a1b2")` |
| `re.split(pattern, s)` | Разделение | `split(r"\W+", "a, b; c")` |
| `re.compile(pattern)` | Компиляция паттерна | |

```python
import re
text = "Мой номер: 123-45-67"
match = re.search(r"\d{3}-\d{2}-\d{2}", text)
if match:
    print(match.group())

phones = re.findall(r"\d+", text)
print(phones)
```

## 8. `collections` — дополнительные структуры

| Тип | Описание | Пример |
|---|---|---|
| `deque` | Двусторонняя очередь | `deque([1, 2, 3])` |
| `Counter` | Подсчёт элементов | `Counter("abracadabra")` |
| `defaultdict` | Словарь со значением по умолчанию | `defaultdict(list)` |
| `OrderedDict` | Упорядоченный словарь (до 3.7) | |
| `namedtuple` | Именованный кортеж | `Point = namedtuple("Point", ["x", "y"])` |

```python
from collections import deque, Counter, defaultdict, namedtuple

# deque
q = deque([1, 2, 3])
q.appendleft(0)
print(q.popleft())

# Counter
cnt = Counter("abracadabra")
print(cnt.most_common(2))

# defaultdict
d = defaultdict(int)
d["a"] += 1

# namedtuple
Point = namedtuple("Point", ["x", "y"])
p = Point(3, 5)
print(p.x, p.y)
```

## 9. `itertools` — итераторы

| Функция | Описание | Пример |
|---|---|---|
| `product(a, b)` | Декартово произведение | `product([1, 2], ["a", "b"])` |
| `permutations(seq)` | Перестановки | `permutations([1, 2, 3], 2)` |
| `combinations(seq, k)` | Сочетания | `combinations("ABCD", 2)` |
| `chain(*iters)` | Объединение итераторов | `chain([1, 2], [3, 4])` |
| `cycle(iter)` | Бесконечный цикл | `cycle([1, 2, 3])` |
| `count(start, step)` | Бесконечный счётчик | `count(0, 2)` |

```python
from itertools import product, permutations, combinations, chain

print(list(product([1, 2], ["a", "b"])))
print(list(permutations([1, 2, 3], 2)))
print(list(combinations("ABCD", 2)))
print(list(chain([1, 2], [3, 4])))
```

## 10. `functools` — функции высшего порядка

| Функция | Описание | Пример |
|---|---|---|
| `reduce(func, seq)` | Накопление значения | `reduce(lambda a, b: a * b, [1, 2, 3, 4])` |
| `lru_cache` | Кеширование результатов | `@lru_cache(maxsize=128)` |
| `partial(func, *args)` | Частичное применение | `partial(pow, 2)` |

```python
from functools import reduce, lru_cache, partial

# reduce
product = reduce(lambda a, b: a * b, [1, 2, 3, 4])
print(product)  # 24

# lru_cache
@lru_cache(maxsize=None)
def fib(n):
    if n <= 1:
        return n
    return fib(n - 1) + fib(n - 2)

print(fib(50))

# partial
pow2 = partial(pow, 2)
print(pow2(3))  # 8
```

## 11. `pathlib` — работа с путями (современный способ)

```python
from pathlib import Path

p = Path("C:/Users/Admin/Documents/file.txt")
print(p.name)        # file.txt
print(p.stem)        # file
print(p.suffix)      # .txt
print(p.parent)      # C:/Users/Admin/Documents
print(p.exists())

# создание папок
Path("new_folder/sub").mkdir(parents=True, exist_ok=True)

# обход
for f in Path(".").iterdir():
    print(f)

# glob
for py in Path(".").glob("**/*.py"):
    print(py)
```

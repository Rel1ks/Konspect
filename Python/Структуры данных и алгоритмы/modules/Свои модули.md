# Создание собственных модулей

**Собственный модуль** — это любой `.py` файл с кодом, который можно импортировать в другие файлы.

## 1. Создание простого модуля

```python
# calc.py
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):
    if b == 0:
        return None
    return a / b

PI = 3.14159
```

```python
# main.py
import calc

print(calc.add(5, 3))
print(calc.PI)
```

## 2. Импорт из своего модуля

```python
# вариант 1
import calc
print(calc.add(2, 3))

# вариант 2
from calc import add, PI
print(add(2, 3))
print(PI)

# вариант 3
from calc import *
print(add(2, 3))
```

## 4. `__name__` и `__main__`

У каждого модуля есть атрибут `__name__`:
- Если файл запущен напрямую — `__name__` = `"__main__"`
- Если файл импортирован — `__name__` = имя модуля

```python
# greetings.py
def hello(name):
    return f"Привет, {name}!"

def goodbye(name):
    return f"Пока, {name}!"

if __name__ == "__main__":
    # тестирование при прямом запуске
    print(hello("Анна"))
    print(goodbye("Анна"))
```

```python
# main.py
import greetings  # ничего не выведет

print(greetings.hello("Борис"))
```

## 5. Создание пакета

**Пакет** — папка с модулями и файлом `__init__.py`.

```
my_package/
├── __init__.py
├── math_ops.py
└── string_ops.py
```

```python
# __init__.py
from .math_ops import add, subtract
from .string_ops import reverse, capitalize

__all__ = ["add", "subtract", "reverse", "capitalize"]
```

```python
# main.py
import my_package

print(my_package.add(5, 3))
print(my_package.reverse("hello"))
```

## 6. `__all__` в модуле

Контролирует, что будет импортировано при `from module import *`.

```python
# shapes.py
__all__ = ["circle_area", "PI"]  # rect_area не импортируется через *

PI = 3.14159

def circle_area(r):
    return PI * r ** 2

def rect_area(a, b):
    return a * b

def _helper():  # приватная (по соглашению)
    pass
```

```python
from shapes import *
print(circle_area(5))
# print(rect_area(2, 3))  # NameError
```

## 7. Приватные функции и атрибуты

По соглашению, имена с `_` в начале считаются внутренними.

```python
# database.py
_connection = None  # "приватная" переменная

def connect():
    global _connection
    _connection = "connected"

def disconnect():
    global _connection
    _connection = None

def _validate(data):  # внутренняя функция
    return isinstance(data, dict)
```

## 8. Документирование модуля

```python
"""
Модуль для работы с числами.

Предоставляет функции для:
- базовых арифметических операций
- проверки чисел
"""

def is_even(n):
    """Проверяет, является ли число чётным."""
    return n % 2 == 0

def is_prime(n):
    """
    Проверяет, является ли число простым.

    Args:
        n: целое положительное число

    Returns:
        True если число простое, иначе False
    """
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True
```

## 9. Относительные импорты внутри пакета

```
my_package/
├── __init__.py
├── utils/
│   ├── __init__.py
│   └── helpers.py
└── core/
    ├── __init__.py
    └── processor.py
```

```python
# core/processor.py
from ..utils.helpers import format_data  # импорт из соседнего пакета
from . import something_else            # импорт из текущего пакета

def process(data):
    return format_data(data)
```

## 10. Практический пример

```python
# file_utils.py
"""
Утилиты для работы с файлами.
"""
import os
import json

def read_text(path):
    with open(path, "r", encoding="utf-8") as f:
        return f.read()

def write_text(path, content):
    with open(path, "w", encoding="utf-8") as f:
        f.write(content)

def read_json(path):
    with open(path, "r", encoding="utf-8") as f:
        return json.load(f)

def write_json(path, data):
    with open(path, "w", encoding="utf-8") as f:
        json.dump(data, f, ensure_ascii=False, indent=2)

def file_size(path):
    return os.path.getsize(path)

def file_exists(path):
    return os.path.exists(path)

if __name__ == "__main__":
    # тест
    write_text("test.txt", "Hello, World!")
    print(read_text("test.txt"))
    os.remove("test.txt")
```

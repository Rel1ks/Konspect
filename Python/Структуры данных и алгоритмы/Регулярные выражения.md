# Регулярные выражения (regex) в Python

**Регулярные выражения** — шаблоны для поиска, замены и извлечения текста. Модуль `re`.

## 1. Основные функции `re`

| Функция | Описание | Пример |
|---|---|---|
| `re.search(pattern, s)` | Ищет первое совпадение (где угодно) | `search(r"\d+", "abc123")` |
| `re.match(pattern, s)` | Совпадение с начала строки | `match(r"\d+", "123abc")` |
| `re.findall(pattern, s)` | Все совпадения (списком) | `findall(r"\d+", "a1b2c3")` |
| `re.finditer(pattern, s)` | Все совпадения (итератор) | |
| `re.sub(pattern, repl, s)` | Замена совпадений | `sub(r"\d+", "X", "a1b2")` |
| `re.split(pattern, s)` | Разделение по шаблону | `split(r"\W+", "a, b; c")` |
| `re.compile(pattern)` | Компиляция шаблона для переиспользования | |

```python
import re

text = "Мой номер: 123-45-67, другой: 98-76-54"

print(re.search(r"\d{2}-\d{2}-\d{2}", text).group())
print(re.findall(r"\d{2}-\d{2}-\d{2}", text))
print(re.sub(r"\d{3}", "XXX", text))
```

## 2. Синтаксис шаблонов

### Базовые символы

| Символ | Значение | Пример |
|---|---|---|
| `.` | Любой символ (кроме `\n`) | `"a.b"` совпадает с "acb", "a b" |
| `\d` | Цифра (0-9) | `\d+` → "123" |
| `\D` | Не цифра | |
| `\w` | Буква, цифра или `_` | `\w+` → "hello_123" |
| `\W` | Не буква/цифра/`_` | |
| `\s` | Пробельный символ (пробел, `\t`, `\n`) | |
| `\S` | Не пробельный символ | |

```python
print(re.findall(r"\d+", "Цена: 100 руб, вес: 25 кг"))
print(re.findall(r"\w+", "Привет, Мир! 2024"))
```

### Квантификаторы (количество повторений)

| Символ | Значение | Пример |
|---|---|---|
| `*` | 0 или больше | `\d*` |
| `+` | 1 или больше | `\d+` |
| `?` | 0 или 1 | `colou?r` → color, colour |
| `{n}` | Ровно n | `\d{3}` |
| `{n,}` | n или больше | `\d{2,}` |
| `{n,m}` | от n до m | `\d{2,4}` |

```python
print(re.findall(r"\d{2,4}", "1 22 333 4444 55555"))
```

### Якоря

| Символ | Значение | Пример |
|---|---|---|
| `^` | Начало строки | `^Hello` |
| `$` | Конец строки | `world$` |
| `\b` | Граница слова | `\bword\b` |
| `\B` | Не граница слова | |

```python
text = "Hello world! Hello everyone"
print(re.findall(r"^Hello", text))
print(re.findall(r"world$", text))
print(re.findall(r"\bHello\b", text))
```

### Классы символов

```python
# [abc] — любой из символов a, b, c
print(re.findall(r"[aeiou]", "hello world"))  # гласные

# [a-z] — диапазон
print(re.findall(r"[a-zA-Z]+", "Hello123World"))

# [^...] — отрицание
print(re.findall(r"[^0-9]", "abc123def"))  # не цифры

# [0-9] — то же что \d
# [a-zA-Z0-9_] — то же что \w
```

### Группы и захват

| Синтаксис | Описание | Пример |
|---|---|---|
| `(pattern)` | Захват группы | `(\d+)` |
| `(?:pattern)` | Незапоминающая группа | `(?:\d+)` |
| `(?P<name>p)` | Именованная группа | `(?P<num>\d+)` |

```python
text = "Анна: 25, Борис: 30"

# захват групп
matches = re.findall(r"(\w+): (\d+)", text)
print(matches)  # [("Анна", "25"), ("Борис", "30")]

for name, age in matches:
    print(f"{name}: {age}")

# именованные группы
match = re.search(r"(?P<name>\w+): (?P<age>\d+)", text)
print(match.group("name"), match.group("age"))
```

### Логическое ИЛИ

```python
print(re.findall(r"кот|собака", "кот и собака и ещё кот"))
print(re.findall(r"(кот|собака) (ест|спит)", "кот ест, собака спит"))
```

## 3. Объект Match

```python
text = "Контакты: +7 (123) 456-78-90"
match = re.search(r"(\+7)\s?\((\d+)\)\s?([\d-]+)", text)

if match:
    print(match.group())      # +7 (123) 456-78-90
    print(match.group(0))     # то же самое
    print(match.group(1))     # +7
    print(match.group(2))     # 123
    print(match.group(3))     # 456-78-90
    print(match.groups())     # ("+7", "123", "456-78-90")
    print(match.start())      # начальный индекс
    print(match.end())        # конечный индекс
```

## 4. Флаги

| Флаг | Сокращение | Описание |
|---|---|---|
| `re.IGNORECASE` | `re.I` | Игнорировать регистр |
| `re.MULTILINE` | `re.M` | `^` и `$` работают для каждой строки |
| `re.DOTALL` | `re.S` | `.` включает `\n` |
| `re.VERBOSE` | `re.X` | Пробелы игнорируются, можно комментарии |
| `re.ASCII` | `re.A` | `\w`, `\d`, `\b` только для ASCII |

```python
text = "Hello\nWorld\nHello"

print(re.findall(r"^hello", text, re.I | re.M))
print(re.findall(r"hello.world", text, re.S))
```

## 5. `re.compile()` — компиляция шаблона

Для многократного использования одного шаблона.

```python
# без compile
for line in lines:
    if re.search(r"^\d+:", line):
        print(line)

# с compile
pattern = re.compile(r"^\d+:")
for line in lines:
    if pattern.search(line):
        print(line)

# compile с флагами
email_pattern = re.compile(r"[\w.]+@[\w.]+\.\w+", re.I)
```

## 6. `re.sub()` — замена

```python
text = "Мой телефон: 123-45-67 и 98-76-54"

# простая замена
print(re.sub(r"\d+", "X", text))

# замена с группами
print(re.sub(r"(\d{2})-(\d{2})-(\d{2})", r"\3-\2-\1", text))

# замена через функцию
def mask(match):
    return "***-" + match.group(2) + "-***"

print(re.sub(r"(\d{2})-(\d{2})-(\d{2})", mask, text))

# ограничение количества замен
print(re.sub(r"\d+", "X", text, count=2))
```

## 7. `re.split()` — разделение

```python
text = "яблоко, груша; банан   апельсин"

print(re.split(r"[,; ]+", text))
print(re.split(r"\W+", "Привет, Мир! Как дела?"))
```

## 8. Lookahead и Lookbehind

| Синтаксис | Название | Описание |
|---|---|---|
| `(?=...)` | Positive lookahead | За чем-то следует ... |
| `(?!...)` | Negative lookahead | За чем-то НЕ следует ... |
| `(?<=...)` | Positive lookbehind | После ... |
| `(?<!...)` | Negative lookbehind | Не после ... |

```python
# positive lookahead — цифры перед "руб"
print(re.findall(r"\d+(?=\s*руб)", "100 руб, 200 дол, 300 руб"))

# negative lookahead — цифры не перед "руб"
print(re.findall(r"\d+(?!\s*руб)", "100 руб, 200 дол, 300 руб"))

# positive lookbehind — после "цена:"
print(re.findall(r"(?<=цена: )\d+", "цена: 500, скидка: 100"))

# negative lookbehind — не после "скидка:"
print(re.findall(r"(?<!скидка: )\d+", "цена: 500, скидка: 100"))
```

## 9. Жадные и ленивые квантификаторы

По умолчанию квантификаторы жадные — захватывают максимум.

```python
text = "<div>Привет</div><span>Мир</span>"

# жадный — захватит всё до последнего >
print(re.findall(r"<.*>", text))

# ленивый — добавить ?
print(re.findall(r"<.*?>", text))

# ленивый +
print(re.findall(r"<.+?>", text))
```

| Жадный | Ленивый |
|---|---|
| `*` | `*?` |
| `+` | `+?` |
| `?` | `??` |
| `{n,m}` | `{n,m}?` |

## 10. Практические примеры

### Валидация email

```python
def is_valid_email(email):
    pattern = r"^[\w.+-]+@[\w-]+\.[\w.]+$"
    return bool(re.match(pattern, email))

print(is_valid_email("user@example.com"))
print(is_valid_email("invalid-email"))
```

### Извлечение URL

```python
text = "Сайты: https://example.com и http://test.org/page"
urls = re.findall(r"https?://[\w./-]+", text)
print(urls)
```

### Поиск всех дат

```python
text = "Даты: 15.01.2024, 01.02, 2024-03-15"
dates = re.findall(r"\d{2}\.\d{2}\.\d{4}", text)
print(dates)
```

### Очистка текста

```python
text = "Hello,  Мир!  Это  тест."
clean = re.sub(r"\s+", " ", text)
print(clean)

# удалить html-теги
html = "<p>Текст <b>жирный</b></p>"
no_tags = re.sub(r"<[^>]+>", "", html)
print(no_tags)
```

### Токенизация

```python
text = "Hello, world! How are you?"
tokens = re.findall(r"\b\w+\b", text)
print(tokens)
```

### Проверка пароля

```python
def check_password(password):
    errors = []
    if len(password) < 8:
        errors.append("Минимум 8 символов")
    if not re.search(r"[A-Z]", password):
        errors.append("Нужна заглавная буква")
    if not re.search(r"[a-z]", password):
        errors.append("Нужна строчная буква")
    if not re.search(r"\d", password):
        errors.append("Нужна цифра")
    if not re.search(r"[!@#$%^&*]", password):
        errors.append("Нужен спецсимвол")
    return errors if errors else "Пароль надёжный"

print(check_password("Pass123!"))
```

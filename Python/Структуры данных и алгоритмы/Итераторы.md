# Итераторы (iterators) в Python

**Итератор** — объект, который возвращает элементы по одному, запоминая текущую позицию. Любой объект, поддерживающий `__iter__()` и `__next__()`.

## 1. Итерируемые объекты и итераторы

| Термин | Описание | Примеры |
|---|---|---|
| **Итерируемый** (iterable) | Можно передать в `for` или `iter()` | list, str, dict, tuple, set, range |
| **Итератор** (iterator) | Возвращает элементы через `next()` | iter(list), генератор, file |

```python
# iterable → iterator
nums = [1, 2, 3]
it = iter(nums)

print(next(it))  # 1
print(next(it))  # 2
print(next(it))  # 3
# print(next(it))  # StopIteration
```

## 2. Как работает цикл `for`

```python
# for автоматически создаёт итератор
for x in [1, 2, 3]:
    print(x)

# это эквивалентно:
it = iter([1, 2, 3])
while True:
    try:
        x = next(it)
        print(x)
    except StopIteration:
        break
```

## 3. Создание своего итератора

Класс должен реализовать `__iter__()` (возвращает `self`) и `__next__()` (возвращает элемент или `StopIteration`).

```python
class Counter:
    def __init__(self, start, end):
        self.current = start
        self.end = end

    def __iter__(self):
        return self

    def __next__(self):
        if self.current > self.end:
            raise StopIteration
        value = self.current
        self.current += 1
        return value

counter = Counter(1, 5)
for n in counter:
    print(n)  # 1, 2, 3, 4, 5
```

## 4. `iter()` с вызываемым объектом

`iter(func, sentinel)` — вызывает функцию до тех пор, пока она не вернёт `sentinel`.

```python
import random

def rand():
    return random.randint(1, 6)

# генерирует числа, пока не выпадет 6
for value in iter(rand, 6):
    print(value)
print("Выпала 6!")
```

## 5. Встроенные функции для работы с итераторами

| Функция | Описание | Пример |
|---|---|---|
| `iter(obj)` | Получить итератор | `iter([1, 2, 3])` |
| `next(it)` | Следующий элемент | `next(it)` |
| `next(it, default)` | Следующий или default | `next(it, "конец")` |
| `enumerate(seq)` | Индекс + элемент | |
| `zip(*iters)` | Объединение итераторов | |
| `map(func, it)` | Применить функцию | |
| `filter(func, it)` | Отфильтровать | |
| `reversed(seq)` | В обратном порядке | |
| `sorted(seq)` | Отсортировать | |

```python
it = iter([10, 20, 30])
print(next(it, "нет"))  # 10
print(next(it, "нет"))  # 20
print(next(it, "нет"))  # 30
print(next(it, "нет"))  # "нет"
```

## 6. Итераторы последовательностей

```python
# enumerate
fruits = ["яблоко", "груша", "банан"]
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# zip
names = ["Анна", "Борис", "Вика"]
ages = [25, 30, 22]
for name, age in zip(names, ages):
    print(f"{name}: {age}")

# map
nums = [1, 2, 3, 4]
squares = map(lambda x: x ** 2, nums)
print(list(squares))

# filter
evens = filter(lambda x: x % 2 == 0, nums)
print(list(evens))
```

## 7. Итераторы итерируют один раз

```python
nums = [1, 2, 3]
it = iter(nums)

print(list(it))  # [1, 2, 3]
print(list(it))  # [] — итератор пуст
```

## 8. `itertools` — мощные итераторы

### Бесконечные итераторы

| Функция | Описание | Пример |
|---|---|---|
| `count(start, step)` | Бесконечный счётчик | `count(0, 2)` |
| `cycle(iter)` | Бесконечный повтор | `cycle([1, 2, 3])` |
| `repeat(x, n)` | Повтор элемента | `repeat(5, 3)` |

```python
from itertools import count, cycle, repeat

for i in count(10, 2):
    if i > 20:
        break
    print(i)

# cycle
c = 0
for x in cycle(["A", "B", "C"]):
    print(x)
    c += 1
    if c > 5:
        break
```

### Комбинаторика

| Функция | Описание | Пример |
|---|---|---|
| `product(*iters)` | Декартово произведение | `product([1, 2], ["a", "b"])` |
| `permutations(seq, r)` | Перестановки | `permutations([1, 2, 3], 2)` |
| `combinations(seq, r)` | Сочетания | `combinations("ABCD", 2)` |
| `combinations_with_replacement(seq, r)` | Сочетания с повторениями | |

```python
from itertools import product, permutations, combinations

print(list(product([1, 2], ["a", "b"])))
print(list(permutations([1, 2, 3], 2)))
print(list(combinations([1, 2, 3], 2)))
```

### Группировка и объединение

| Функция | Описание | Пример |
|---|---|---|
| `chain(*iters)` | Объединить итераторы | `chain([1, 2], [3, 4])` |
| `groupby(iter, key)` | Группировка по ключу | `groupby("aaabbbcca")` |
| `accumulate(iter)` | Накопление (сумма и т.д.) | `accumulate([1, 2, 3])` |

```python
from itertools import chain, groupby, accumulate

print(list(chain([1, 2], [3, 4])))

for key, group in groupby("aaabbbcca"):
    print(key, list(group))

print(list(accumulate([1, 2, 3, 4])))  # [1, 3, 6, 10]
```

### Фильтрация

| Функция | Описание | Пример |
|---|---|---|
| `takewhile(pred, it)` | Брать пока истинно | `takewhile(lambda x: x < 5, [1, 3, 7, 2])` |
| `dropwhile(pred, it)` | Пропустить пока истинно | `dropwhile(lambda x: x < 5, [1, 3, 7, 2])` |
| `filterfalse(pred, it)` | Обратный filter | `filterfalse(None, [0, 1, 0])` |
| `compress(data, sel)` | Селектор True/False | `compress("ABCD", [1, 0, 1])` |
| `islice(it, start, stop, step)` | Срез итератора | `islice(range(10), 2, 8, 2)` |

```python
from itertools import takewhile, dropwhile, filterfalse, compress, islice

print(list(takewhile(lambda x: x < 5, [1, 3, 7, 2])))
print(list(dropwhile(lambda x: x < 5, [1, 3, 7, 2])))
print(list(compress("ABCD", [1, 0, 1, 0])))
print(list(islice(range(10), 2, 8, 2)))
```

## 9. Итераторы в стандартной библиотеке

```python
# reversed
for x in reversed([1, 2, 3]):
    print(x)

# sorted (возвращает список, не итератор)

with open("file.txt") as f:
```

## 10. Практические примеры

### Пагинация (по chunk'ам)

```python
from itertools import islice

def chunks(iterable, size):
    it = iter(iterable)
    return iter(lambda: tuple(islice(it, size)), ())

data = range(10)
for chunk in chunks(data, 3):
    print(chunk)
```

### Обход вложенных структур

```python
def flatten(nested):
    for item in nested:
        if isinstance(item, (list, tuple)):
            yield from flatten(item)
        else:
            yield item

nested = [1, [2, [3, 4], 5], 6]
print(list(flatten(nested)))  # [1, 2, 3, 4, 5, 6]
```

### Бесконечная последовательность

```python
def fibonacci():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

fib = fibonacci()
for _ in range(10):
    print(next(fib))
```

### Объединение нескольких итераторов

```python
def interleave(*iterables):
    iterators = [iter(it) for it in iterables]
    while iterators:
        for it in iterators[:]:
            try:
                yield next(it)
            except StopIteration:
                iterators.remove(it)

print(list(interleave([1, 2, 3], ["a", "b", "c"])))
```

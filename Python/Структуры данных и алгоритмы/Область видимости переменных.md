# Область видимости переменных (variable scope)

**Область видимости** определяет, где в коде доступна переменная. Python использует правило **LEGB**.

## 1. LEGB — правило поиска

Python ищет имя переменной в порядке:

| Уровень | Название | Описание |
|---|---|---|
| **L** | **Local** | Локальная — внутри текущей функции |
| **E** | **Enclosing** | Внешняя — во внешней функции (для замыканий) |
| **G** | **Global** | Глобальная — на уровне модуля |
| **B** | **Built-in** | Встроенная — встроенные имена Python |

```python
x = "глобальная"  # Global

def outer():
    x = "внешняя"  # Enclosing

    def inner():
        x = "локальная"  # Local
        print(x)

    inner()

outer()  # локальная
```

## 2. Локальная область (Local)

Переменные, созданные внутри функции, локальны.

```python
def func():
    y = 10  # локальная
    print(y)

func()
# print(y)  # NameError — y не доступна снаружи
```

## 3. Глобальная область (Global)

Переменные, созданные вне функций, глобальны.

```python
x = 5  # глобальная

def func():
    print(x)  # читать глобальную можно

func()
print(x)
```

## 4. Ключевое слово `global`

Чтобы **изменить** глобальную переменную внутри функции, нужно объявить `global`.

```python
counter = 0

def increment():
    global counter
    counter += 1

increment()
increment()
print(counter)  # 2
```

Без `global` создастся новая локальная переменная:

```python
counter = 0

def increment():
    counter = 1  # локальная, не глобальная

increment()
print(counter)  # 0
```

## 5. Ключевое слово `nonlocal`

`nonlocal` используется во вложенных функциях для изменения переменной из внешней (enclosing) функции.

```python
def outer():
    x = 10

    def inner():
        nonlocal x
        x += 5

    inner()
    print(x)  # 15

outer()
```

```python
# без nonlocal — создаётся локальная
def outer():
    x = 10

    def inner():
        x = 20  # локальная внутри inner

    inner()
    print(x)  # 10
```

## 6. Примеры LEGB

```python
# Built-in
print(len([1, 2, 3]))  # len — из built-in

# Global
x = 10

def outer():
    # Enclosing
    x = 20

    def inner():
        # Local
        x = 30
        print(x)  # 30

    inner()
    print(x)  # 20

outer()
print(x)  # 10
```

## 7. Изменение глобального списка/словаря без `global`

Изменять содержимое изменяемых объектов можно без `global`.

```python
items = [1, 2, 3]

def add_item(x):
    items.append(x)  # изменяем содержимое, не переприсваиваем

add_item(4)
print(items)  # [1, 2, 3, 4]
```

```python
# а вот переприсваивание требует global
items = [1, 2, 3]

def reset():
    global items
    items = []  # переприсваивание

reset()
print(items)  # []
```

## 8. Область видимости в классах

```python
class MyClass:
    class_var = "класс"  # атрибут класса

    def __init__(self):
        self.instance_var = "экземпляр"  # атрибут экземпляра

    def method(self):
        local_var = "локальная"  # локальная в методе
        print(local_var)
        print(self.instance_var)
        print(MyClass.class_var)
```

## 9. Область видимости в comprehensions

В Python 3 переменная цикла в comprehension **локальна** (в Python 2 была глобальной).

```python
x = "глобальная"
squares = [x ** 2 for x in range(5)]  # x — локальна для comprehension
print(x)  # "глобальная" — не изменилась
```

## 10. `globals()` и `locals()`

```python
x = 10

def func():
    y = 20
    print("locals:", locals())
    print("globals:", globals().keys())

func()
```

## 11. Типичные ошибки

### Использование переменной до присваивания

```python
x = 5

def func():
    print(x)  # UnboundLocalError
    x = 10

func()
```

Python видит присваивание `x = 10` и считает `x` локальной, но `print(x)` вызывается до присваивания.

### Решение:

```python
x = 5

def func():
    global x
    print(x)
    x = 10
```

## 12. Замыкания и область видимости

Внутренняя функция запоминает переменные внешней, даже после завершения внешней функции.

```python
def make_counter():
    count = 0  # enclosing

    def counter():
        nonlocal count
        count += 1
        return count

    return counter

c1 = make_counter()
c2 = make_counter()

print(c1())  # 1
print(c1())  # 2
print(c2())  # 1 (свой счётчик)
```

## 13. Практические советы

```python
# 1. Читать глобальные переменные можно без global
x = 10
def show():
    print(x)

# 2. Изменять глобальные изменяемые объекты — тоже
data = []
def add(value):
    data.append(value)

# 3. Переприсваивание глобальной переменной — только через global
x = 10
def set_x(val):
    global x
    x = val

# 4. Для enclosing — nonlocal
def outer():
    x = 10
    def inner():
        nonlocal x
        x = 20
    inner()
    print(x)
```

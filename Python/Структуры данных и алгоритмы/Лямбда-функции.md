# Лямбда-функции (lambda) в Python

**Лямбда-функция** — анонимная функция, определённая в одной строке. Используется для простых операций, где не нужно писать полноценное `def`.

## 1. Синтаксис

| Синтаксис | Описание |
|---|---|
| `lambda args: expression` | Принимает аргументы, возвращает результат выражения |

```python
# обычная функция
def square(x):
    return x ** 2

# лямбда
square = lambda x: x ** 2

print(square(5))  # 25
```

## 2. Лямбда с разным количеством аргументов

```python
# без аргументов
f = lambda: "Hello"
print(f())

# один аргумент
double = lambda x: x * 2

# два аргумента
add = lambda a, b: a + b

# много аргументов
sum_all = lambda *args: sum(args)

# с именованными
info = lambda name, age: f"{name}: {age} лет"

print(add(3, 5))
print(sum_all(1, 2, 3, 4))
print(info("Анна", 25))
```

## 3. Лямбда с условием

```python
# тернарный оператор внутри лямбды
even_or_odd = lambda x: "чётное" if x % 2 == 0 else "нечётное"

print(even_or_odd(4))  # чётное
print(even_or_odd(7))  # нечётное

# вложенные условия
check = lambda x: "A" if x > 90 else ("B" if x > 75 else "C")
print(check(85))  # B
```

## 4. Использование с `map()`

`map(func, iterable)` — применяет функцию к каждому элементу.

```python
nums = [1, 2, 3, 4, 5]

squares = list(map(lambda x: x ** 2, nums))
print(squares)  # [1, 4, 9, 16, 25]

# с несколькими списками
a = [1, 2, 3]
b = [4, 5, 6]
sums = list(map(lambda x, y: x + y, a, b))
print(sums)  # [5, 7, 9]
```

## 5. Использование с `filter()`

`filter(func, iterable)` — оставляет элементы, для которых функция вернула `True`.

```python
nums = [1, 2, 3, 4, 5, 6]

evens = list(filter(lambda x: x % 2 == 0, nums))
print(evens)  # [2, 4, 6]

# фильтр строк
words = ["кот", "собака", "дом", "автомобиль"]
long_words = list(filter(lambda w: len(w) > 4, words))
print(long_words)  # ["собака", "автомобиль"]
```

## 6. Использование с `sorted()`

Сортировка с кастомным ключом.

```python
# сортировка по длине строки
words = ["яблоко", "дом", "кот", "автомобиль", "море"]
sorted_by_len = sorted(words, key=lambda w: len(w))
print(sorted_by_len)

# сортировка по последней букве
sorted_by_last = sorted(words, key=lambda w: w[-1])
print(sorted_by_last)

# сортировка списка кортежей по второму элементу
pairs = [("Анна", 25), ("Борис", 30), ("Вика", 22)]
sorted_by_age = sorted(pairs, key=lambda p: p[1])
print(sorted_by_age)

# по убыванию
sorted_desc = sorted(pairs, key=lambda p: p[1], reverse=True)
print(sorted_desc)
```

## 7. Использование с `min()` и `max()`

```python
# поиск по критерию
people = [("Анна", 25), ("Борис", 30), ("Вика", 22)]

youngest = min(people, key=lambda p: p[1])
oldest = max(people, key=lambda p: p[1])

print(youngest)  # ("Вика", 22)
print(oldest)    # ("Борис", 30)

# самая короткая строка
words = ["яблоко", "дом", "кот"]
shortest = min(words, key=len)
print(shortest)  # "кот"
```

## 8. Лямбда в словарях

```python
# калькулятор через словарь
operations = {
    "add": lambda a, b: a + b,
    "sub": lambda a, b: a - b,
    "mul": lambda a, b: a * b,
    "div": lambda a, b: a / b if b != 0 else None,
}

print(operations["add"](5, 3))
print(operations["div"](10, 2))
```

## 9. Лямбда в списках

```python
# список функций
funcs = [
    lambda x: x * 2,
    lambda x: x ** 2,
    lambda x: x + 5,
]

for f in funcs:
    print(f(10))
```

## 10. Лямбда с `reduce()`

```python
from functools import reduce

nums = [1, 2, 3, 4, 5]
product = reduce(lambda a, b: a * b, nums)
print(product)  # 120

sum_all = reduce(lambda a, b: a + b, nums)
print(sum_all)  # 15
```

## 11. Лямбда в декораторах

```python
def repeat(n):
    return lambda func: lambda *args, **kwargs: [func(*args, **kwargs) for _ in range(n)]

@repeat(3)
def greet(name):
    return f"Привет, {name}!"

print(greet("Анна"))  # ["Привет, Анна!", "Привет, Анна!", "Привет, Анна!"]
```

## 12. Ограничения лямбда

Лямбда ограничена **одним выражением**. Нельзя использовать:
- `return`, `yield`
- Циклы (`for`, `while`)
- Многострочный код
- Аннотации типов

```python
# так нельзя:
# lambda x: if x > 0: return x  # SyntaxError

# используйте обычную функцию:
def process(x):
    if x > 0:
        return x
    return 0

# лямбда с тернарным оператором — можно
process = lambda x: x if x > 0 else 0
```

## 13. Замена лямбды обычной функцией

Когда код становится сложным — лучше использовать `def`.

```python
# плохо
calc = lambda op, a, b: (
    a + b if op == "add" else
    a - b if op == "sub" else
    a * b if op == "mul" else
    None
)

# хорошо
def calc(op, a, b):
    if op == "add": return a + b
    if op == "sub": return a - b
    if op == "mul": return a * b
    return None
```

## 14. Практические примеры

```python
# сортировка файлов по расширению
files = ["doc.txt", "image.png", "data.csv", "script.py"]
by_ext = sorted(files, key=lambda f: f.split(".")[-1])
print(by_ext)

# извлечение ключей из списка словарей
users = [
    {"name": "Анна", "age": 25},
    {"name": "Борис", "age": 30},
    {"name": "Вика", "age": 22},
]
names = list(map(lambda u: u["name"], users))
print(names)

# проверка условий
nums = [-5, 3, -1, 0, 7, -2]
positive = list(filter(lambda x: x > 0, nums))
print(positive)

# преобразование данных
prices = ["100", "250", "50", "300"]
total = sum(map(lambda p: int(p), prices))
print(total)
```

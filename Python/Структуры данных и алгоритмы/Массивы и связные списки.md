# Массивы и связные списки (Arrays & Linked Lists)

## Массивы (Array)

**Массив** — последовательность элементов одного типа, хранящаяся в непрерывной области памяти.

### Характеристики

| Свойство | Описание |
|---|---|
| Память | Непрерывный блок |
| Доступ по индексу | O(1) — мгновенный |
| Поиск значения | O(n) — линейный |
| Вставка в начало/середину | O(n) — сдвиг элементов |
| Удаление из начала/середины | O(n) — сдвиг элементов |
| Вставка в конец | O(1) — если есть место |
| Размер | Фиксированный (в классическом массиве) |

### Статический массив (фиксированный размер)

```python
# в Python нет статических массивов, но есть array из модуля array
from array import array

# типизированный массив (все элементы одного типа)
arr = array("i", [1, 2, 3, 4, 5])
print(arr[0])  # O(1)
arr[2] = 10
print(arr)

arr.append(6)  # O(1) амортизированно
```

### Динамический массив (list в Python)

Список в Python — динамический массив, который автоматически расширяется.

```python
nums = [1, 2, 3, 4, 5]

# доступ по индексу — O(1)
print(nums[0])
print(nums[-1])

# поиск — O(n)
print(nums.index(3))

# вставка в конец — O(1) амортизированно
nums.append(6)

# вставка в начало/середину — O(n)
nums.insert(0, 0)

# удаление с конца — O(1)
nums.pop()

# удаление из начала/середины — O(n)
nums.pop(0)
nums.remove(3)
```

### Преимущества и недостатки

| ✅ Преимущества | ❌ Недостатки |
|---|---|
| Быстрый доступ по индексу O(1) | Вставка/удаление не в конце — O(n) |
| Кэш-локальность (память рядом) | Требует непрерывный блок памяти |
| Эффективен для чтения | Расширение — копирование при переполнении |
| Простота реализации | Фиксированный тип в статическом |

## Связные списки (Linked List)

**Связный список** — последовательность узлов, где каждый узел содержит данные и ссылку на следующий узел.

### Односвязный список (Singly Linked List)

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None
        self.size = 0

    def append(self, data):
        """Добавление в конец — O(n)"""
        new_node = Node(data)
        if not self.head:
            self.head = new_node
        else:
            current = self.head
            while current.next:
                current = current.next
            current.next = new_node
        self.size += 1

    def prepend(self, data):
        """Добавление в начало — O(1)"""
        new_node = Node(data)
        new_node.next = self.head
        self.head = new_node
        self.size += 1

    def insert(self, index, data):
        """Вставка по индексу — O(n)"""
        if index == 0:
            self.prepend(data)
            return
        new_node = Node(data)
        current = self.head
        for _ in range(index - 1):
            if not current:
                raise IndexError("Выход за границы")
            current = current.next
        new_node.next = current.next
        current.next = new_node
        self.size += 1

    def delete(self, data):
        """Удаление по значению — O(n)"""
        if not self.head:
            return
        if self.head.data == data:
            self.head = self.head.next
            self.size -= 1
            return
        current = self.head
        while current.next:
            if current.next.data == data:
                current.next = current.next.next
                self.size -= 1
                return
            current = current.next

    def get(self, index):
        """Доступ по индексу — O(n)"""
        current = self.head
        for _ in range(index):
            if not current:
                raise IndexError
            current = current.next
        return current.data if current else None

    def __len__(self):
        return self.size

    def __repr__(self):
        values = []
        current = self.head
        while current:
            values.append(str(current.data))
            current = current.next
        return " -> ".join(values) + " -> None"

# использование
ll = LinkedList()
ll.append(1)
ll.append(2)
ll.prepend(0)
ll.insert(2, 1.5)
print(ll)       # 0 -> 1 -> 1.5 -> 2 -> None
print(ll.get(2))  # 1.5
ll.delete(1.5)
print(ll)       # 0 -> 1 -> 2 -> None
```

### Двусвязный список (Doubly Linked List)

Узел содержит ссылки и на следующий, и на предыдущий узел.

```python
class DNode:
    def __init__(self, data):
        self.data = data
        self.next = None
        self.prev = None

class DoublyLinkedList:
    def __init__(self):
        self.head = None
        self.tail = None
        self.size = 0

    def append(self, data):
        """Добавление в конец — O(1)"""
        new_node = DNode(data)
        if not self.head:
            self.head = self.tail = new_node
        else:
            new_node.prev = self.tail
            self.tail.next = new_node
            self.tail = new_node
        self.size += 1

    def prepend(self, data):
        """Добавление в начало — O(1)"""
        new_node = DNode(data)
        if not self.head:
            self.head = self.tail = new_node
        else:
            new_node.next = self.head
            self.head.prev = new_node
            self.head = new_node
        self.size += 1

    def delete(self, node):
        """Удаление узла — O(1)"""
        if node.prev:
            node.prev.next = node.next
        else:
            self.head = node.next
        if node.next:
            node.next.prev = node.prev
        else:
            self.tail = node.prev
        self.size -= 1

    def __repr__(self):
        values = []
        current = self.head
        while current:
            values.append(str(current.data))
            current = current.next
        return " <-> ".join(values) + " <-> None"

dll = DoublyLinkedList()
dll.append(1)
dll.append(2)
dll.prepend(0)
print(dll)  # 0 <-> 1 <-> 2 <-> None
```

### Характеристики связных списков

| Операция | Односвязный | Двусвязный |
|---|---|---|
| Доступ по индексу | O(n) | O(n) |
| Поиск значения | O(n) | O(n) |
| Вставка в начало | O(1) | O(1) |
| Вставка в конец | O(n) (с tail — O(1)) | O(1) |
| Вставка в середину | O(n) | O(n) |
| Удаление из начала | O(1) | O(1) |
| Удаление с конца | O(n) | O(1) |
| Память на узел | data + 1 ссылка | data + 2 ссылки |

### Преимущества и недостатки

| ✅ Преимущества | ❌ Недостатки |
|---|---|
| Вставка/удаление в начало — O(1) | Нет прямого доступа по индексу |
| Нет проблемы непрерывного блока | Дополнительная память на ссылки |
| Динамический размер (без копирования) | Кэш-промахи (память не continuous) |
| Простота объединения/разделения | Сложность реализации |

## Сравнение массивов и связных списков

| Критерий | Массив (list) | Связный список |
|---|---|---|
| Доступ по индексу | O(1) | O(n) |
| Вставка в начало | O(n) | O(1) |
| Вставка в конец | O(1) | O(1) (с tail) |
| Удаление из начала | O(n) | O(1) |
| Поиск | O(n) | O(n) |
| Память | Меньше на элемент | Больше (ссылки) |
| Кэш-локальность | Хорошая | Плохая |

### Когда что использовать

```python
# массив — когда нужен быстрый доступ по индексу
scores = [85, 92, 78, 90]
print(scores[2])  # O(1)

# связный список — когда много вставок/удалений в начале
# (в Python можно использовать deque из collections)
from collections import deque

queue = deque()
queue.appendleft(1)   # O(1)
queue.appendleft(2)   # O(1)
queue.pop()           # O(1)
```

### Пример задачи: стек и очередь

```python
# стек через список (массив) — отлично
stack = []
stack.append(1)
stack.append(2)
stack.pop()  # O(1)

# очередь через список — плохо (pop(0) — O(n))
queue = []
queue.append(1)
queue.append(2)
queue.pop(0)  # O(n) — сдвиг всех элементов

# очередь через deque — хорошо
from collections import deque
queue = deque()
queue.append(1)
queue.append(2)
queue.popleft()  # O(1)
```

# Словари (dict) в Python

**Словарь** — неупорядоченная (до 3.6) / упорядоченная (с 3.7+) коллекция пар «ключ — значение». Ключи должны быть хешируемыми (неизменяемыми).

## 1. Создание

| Способ | Пример |
|---|---|
| Фигурные скобки | `{"a": 1, "b": 2}` |
| `dict()` | `dict(a=1, b=2)` |
| Из пар | `dict([("a", 1), ("b", 2)])` |
| `fromkeys` | `dict.fromkeys(["a", "b"], 0)` |

```python
empty = {}
person = {"name": "Анна", "age": 25, "city": "Москва"}
d = dict(a=1, b=2, c=3)
pairs = dict([("x", 10), ("y", 20)])
keys = dict.fromkeys(["a", "b", "c"], 0)
```

## 2. Доступ к элементам

| Способ | Синтаксис | Описание |
|---|---|---|
| По ключу | `d[key]` | Ошибка, если ключа нет |
| `get` | `d.get(key)` | `None`, если ключа нет |
| `get` с default | `d.get(key, default)` | default, если ключа нет |
| `setdefault` | `d.setdefault(key, value)` | Возвращает значение; если ключа нет — вставляет |

```python
d = {"a": 1, "b": 2}

print(d["a"])
# print(d["z"])  # KeyError

print(d.get("b"))      # 2
print(d.get("z"))      # None
print(d.get("z", 0))   # 0

d.setdefault("c", 3)   # добавит "c": 3
d.setdefault("a", 99)  # не изменит — ключ уже есть
```

## 3. Добавление и обновление

| Синтаксис | Описание |
|---|---|
| `d[key] = value` | Добавить или обновить |
| `d.update(other)` | Обновить из другого словаря/пар |
| `d.update(key=value)` | Обновить через именованные аргументы |

```python
d = {"a": 1}

d["b"] = 2
d["a"] = 100
d.update({"c": 3, "d": 4})
d.update(e=5)

print(d)  # {"a": 100, "b": 2, "c": 3, "d": 4, "e": 5}
```

## 4. Удаление

| Метод | Описание | Пример |
|---|---|---|
| `del d[key]` | Удалить по ключу | `del d["a"]` |
| `d.pop(key)` | Удалить и вернуть значение | `d.pop("a")` |
| `d.pop(key, default)` | С значением по умолчанию | `d.pop("z", None)` |
| `d.popitem()` | Удалить последнюю пару (3.7+) | `d.popitem()` |
| `d.clear()` | Очистить словарь | `d.clear()` |

```python
d = {"a": 1, "b": 2, "c": 3}

del d["c"]
val = d.pop("b")
last = d.popitem()

print(d)
```

## 5. Обход словаря

| Способ | Описание | Пример |
|---|---|---|
| `for key in d:` | По ключам | `for k in d:` |
| `d.keys()` | Ключи | `for k in d.keys():` |
| `d.values()` | Значения | `for v in d.values():` |
| `d.items()` | Пары (ключ, значение) | `for k, v in d.items():` |

```python
d = {"a": 1, "b": 2, "c": 3}

for key in d:
    print(key, d[key])

for key, value in d.items():
    print(f"{key}: {value}")

for value in d.values():
    print(value)
```

## 6. Проверки

| Операция | Синтаксис | Пример |
|---|---|---|
| Есть ли ключ | `in` | `"a" in d` |
| Нет ли ключа | `not in` | `"z" not in d` |
| Длина | `len(d)` | `len(d)` |

```python
d = {"a": 1, "b": 2}
print("a" in d)    # True
print("z" not in d)  # True
print(len(d))      # 2
```

## 7. Слияние словарей (3.9+)

| Способ | Пример |
|---|---|
| `\|` | `merged = a \| b` |
| `\|=` | `a \|= b` |

```python
a = {"x": 1, "y": 2}
b = {"y": 99, "z": 3}

merged = a | b
print(merged)  # {"x": 1, "y": 99, "z": 3}

a |= {"w": 0}
print(a)       # {"x": 1, "y": 2, "w": 0}
```

## 8. Dict comprehension

```python
squares = {x: x**2 for x in range(5)}
print(squares)  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 25}

even_squares = {x: x**2 for x in range(10) if x % 2 == 0}
print(even_squares)  # {0: 0, 2: 4, 4: 16, 6: 36, 8: 64}

# инвертирование
original = {"a": 1, "b": 2, "c": 1}
inverted = {v: k for k, v in original.items()}
print(inverted)  # {1: "c", 2: "b"} — последний ключ побеждает
```

## 9. Копирование

```python
d = {"a": 1, "b": [1, 2, 3]}

# поверхностная копия
d2 = d.copy()
d3 = dict(d)

# глубокая копия
import copy
d4 = copy.deepcopy(d)
```

## 10. Вложенные словари

```python
users = {
    "anna": {"age": 25, "city": "Москва"},
    "boris": {"age": 30, "city": "СПб"}
}

print(users["anna"]["city"])
print(users.get("boris", {}).get("age"))

# обход
for username, info in users.items():
    print(f"{username}: {info['age']} лет, {info['city']}")
```

## 11. Подсчёт элементов (Counter)

```python
# вручную через dict
text = "abracadabra"
counter = {}
for ch in text:
    counter[ch] = counter.get(ch, 0) + 1
print(counter)

# через collections.Counter
from collections import Counter
counter = Counter(text)
print(counter)
print(counter.most_common(2))
```

## 12. Default dict

Автоматически создаёт значение для отсутствующего ключа.

```python
from collections import defaultdict

# список по умолчанию
groups = defaultdict(list)
groups["a"].append(1)
groups["a"].append(2)
groups["b"].append(3)
print(dict(groups))

# число по умолчанию
counter = defaultdict(int)
for ch in "abracadabra":
    counter[ch] += 1
print(dict(counter))
```

## 13. Сортировка словаря

```python
d = {"b": 3, "a": 1, "c": 2}

# по ключам
sorted_by_key = dict(sorted(d.items()))
print(sorted_by_key)

# по значениям
sorted_by_value = dict(sorted(d.items(), key=lambda item: item[1]))
print(sorted_by_value)

# по убыванию
sorted_desc = dict(sorted(d.items(), reverse=True))
print(sorted_desc)
```

## 14. Практические примеры

### Группировка элементов

```python
words = ["яблоко", "груша", "арбуз", "ананас", "абрикос"]
by_first = {}

for word in words:
    first = word[0]
    if first not in by_first:
        by_first[first] = []
    by_first[first].append(word)

print(by_first)
```

### Подсчёт частоты слов

```python
text = "кот собака кот птица собака кот"
words = text.split()

freq = {}
for word in words:
    freq[word] = freq.get(word, 0) + 1

print(freq)
```

### Объединение двух списков в словарь

```python
keys = ["a", "b", "c"]
values = [1, 2, 3]

d = dict(zip(keys, values))
print(d)
```

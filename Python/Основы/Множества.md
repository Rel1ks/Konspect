# Множества (set) в Python

**Множество** — неупорядоченная коллекция уникальных элементов. Поддерживает математические операции над множествами.

## 1. Создание

| Способ | Пример |
|---|---|
| Фигурные скобки | `{1, 2, 3}` |
| `set()` | `set("abracadabra")` → `{"a", "b", "r", "c", "d"}` |
| `set.fromkeys()` | `set.fromkeys([1, 2, 3])` |

```python
empty = set()   # {} — это пустой словарь!
nums = {1, 2, 3, 2, 1}
print(nums)     # {1, 2, 3}

unique = set("hello")
print(unique)   # {'h', 'e', 'l', 'o'}
```

## 2. Добавление и удаление

| Метод | Описание | Пример |
|---|---|---|
| `s.add(x)` | Добавить элемент | `s.add(4)` |
| `s.remove(x)` | Удалить (ошибка если нет) | `s.remove(2)` |
| `s.discard(x)` | Удалить (без ошибки) | `s.discard(10)` |
| `s.pop()` | Удалить и вернуть произвольный | `s.pop()` |
| `s.clear()` | Очистить множество | `s.clear()` |

```python
s = {1, 2, 3}
s.add(4)
s.remove(2)
s.discard(10)  # нет ошибки
print(s)
```

## 3. Проверки

| Операция | Синтаксис | Пример |
|---|---|---|
| Проверка вхождения | `in` | `2 in {1, 2, 3}` → True |
| Длина | `len(s)` | `len({1, 2, 3})` → 3 |
| Минимум | `min(s)` | `min({3, 1, 2})` → 1 |
| Максимум | `max(s)` | `max({3, 1, 2})` → 3 |
| Сумма | `sum(s)` | `sum({1, 2, 3})` → 6 |

## 4. Операции над множествами

### Объединение, пересечение, разность

| Операция | Оператор | Метод | Описание |
|---|---|---|---|
| Объединение | `\|` | `s.union(other)` | Все элементы из обоих |
| Пересечение | `&` | `s.intersection(other)` | Общие элементы |
| Разность | `-` | `s.difference(other)` | Элементы s без other |
| Симметрическая разность | `^` | `s.symmetric_difference(other)` | Элементы в одном, но не в обоих |

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a | b)  # {1, 2, 3, 4, 5, 6}
print(a & b)  # {3, 4}
print(a - b)  # {1, 2}
print(a ^ b)  # {1, 2, 5, 6}
```

### Изменяющие операции (in-place)

| Оператор | Метод | Описание |
|---|---|---|
| `\|=` | `s.update(other)` | Добавить все из other |
| `&=` | `s.intersection_update(other)` | Оставить только общие |
| `-=` | `s.difference_update(other)` | Удалить элементы other |
| `^=` | `s.symmetric_difference_update(other)` | Симметрическая разность |

## 5. Сравнение множеств

| Операция | Оператор | Метод | Описание |
|---|---|---|---|
| Подмножество | `<=` | `s.issubset(other)` | Все элементы s есть в other |
| Строгое подмножество | `<` | — | s ⊆ other и s ≠ other |
| Надмножество | `>=` | `s.issuperset(other)` | Все элементы other есть в s |
| Строгое надмножество | `>` | — | s ⊇ other и s ≠ other |

```python
a = {1, 2, 3}
b = {1, 2, 3, 4, 5}

print(a <= b)  # True
print(b >= a)  # True
print(a < b)   # True
print(a == {3, 2, 1})  # True (порядок не важен)
```

## 6. Неизменяемое множество — `frozenset`

`frozenset` — неизменяемая версия `set`. Можно использовать как ключ словаря.

```python
fs = frozenset([1, 2, 3])

# fs.add(4)     # AttributeError
# fs.remove(1)  # AttributeError

d = {frozenset([1, 2]): "значение"}
print(d)
```

## 7. Генератор множества (set comprehension)

```python
squares = {x**2 for x in range(10)}
print(squares)

evens = {x for x in range(20) if x % 2 == 0}
print(evens)
```

## 8. Удаление дубликатов из списка

```python
nums = [1, 2, 2, 3, 3, 3, 4]
unique = list(set(nums))
print(unique)  # [1, 2, 3, 4]
```

## 9. Практические примеры

### Общие элементы двух списков

```python
a = [1, 2, 3, 4, 5]
b = [4, 5, 6, 7, 8]

common = set(a) & set(b)
print(common)  # {4, 5}
```

### Уникальные элементы из последовательности

```python
text = "abracadabra"
unique_letters = set(text)
print(unique_letters)
print(sorted(unique_letters))
```

### Проверка на уникальность

```python
def has_duplicates(lst):
    return len(lst) != len(set(lst))

print(has_duplicates([1, 2, 3]))    # False
print(has_duplicates([1, 2, 2]))    # True
```

## 10. Неупорядоченность

Множества не гарантируют порядок элементов.

```python
s = {3, 1, 2}
print(s)  # может быть {1, 2, 3} или {3, 1, 2} — порядок не гарантирован
```

Для有序的 уникальных элементов используйте `sorted(set(...))`.

## 11. Элементы множества

Элементами множества могут быть только **хешируемые** (неизменяемые) объекты.

```python
# можно: числа, строки, кортежи
s = {1, "text", (1, 2)}

# нельзя: списки, словари, множества
# s = {[1, 2]}  # TypeError
```

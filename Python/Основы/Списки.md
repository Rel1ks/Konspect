# Списки (list) в Python

**Список** — изменяемая упорядоченная коллекция элементов. Может содержать элементы разных типов.

## 1. Создание

| Способ | Пример |
|---|---|
| Квадратные скобки | `[1, 2, 3]` |
| `list()` | `list("abc")` → `["a", "b", "c"]` |
| Генератор списка | `[x**2 for x in range(5)]` |

```python
empty = []
nums = [1, 2, 3, 4, 5]
mixed = [1, "text", True, 3.14]
chars = list("hello")
```

## 2. Индексация и срезы

```python
nums = [10, 20, 30, 40, 50]

print(nums[0])      # 10
print(nums[-1])     # 50
print(nums[1:4])    # [20, 30, 40]
print(nums[::2])    # [10, 30, 50]
print(nums[::-1])   # [50, 40, 30, 20, 10]
```

## 3. Основные методы

### Добавление и удаление

| Метод | Описание | Пример |
|---|---|---|
| `list.append(x)` | Добавить в конец | `a.append(4)` |
| `list.extend(seq)` | Расширить списком | `a.extend([5, 6])` |
| `list.insert(i, x)` | Вставить по индексу | `a.insert(0, 0)` |
| `list.remove(x)` | Удалить первое вхождение | `a.remove(2)` |
| `list.pop(i)` | Удалить и вернуть по индексу | `a.pop()` |
| `list.clear()` | Очистить список | `a.clear()` |

```python
a = [1, 2, 3]
a.append(4)
a.insert(0, 0)
a.extend([5, 6])
a.remove(2)
last = a.pop()
print(a)
```

### Поиск и подсчёт

| Метод | Описание | Пример |
|---|---|---|
| `list.index(x)` | Индекс первого вхождения | `a.index(3)` |
| `list.count(x)` | Количество вхождений | `a.count(2)` |

```python
a = [1, 2, 2, 3, 2]
print(a.index(3))
print(a.count(2))
```

### Сортировка и переворот

| Метод | Описание | Пример |
|---|---|---|
| `list.sort()` | Сортировка in-place | `a.sort()` |
| `list.sort(reverse=True)` | По убыванию | `a.sort(reverse=True)` |
| `list.reverse()` | Переворот in-place | `a.reverse()` |

```python
a = [3, 1, 4, 1, 5]
a.sort()
print(a)

a.reverse()
print(a)
```

## 4. Операции со списками

| Операция | Синтаксис | Пример |
|---|---|---|
| Длина | `len(lst)` | `len([1, 2, 3])` → 3 |
| Конкатенация | `+` | `[1, 2] + [3, 4]` |
| Повторение | `*` | `[0] * 3` → `[0, 0, 0]` |
| Проверка | `in` | `2 in [1, 2, 3]` → True |
| Минимум | `min(lst)` | `min([3, 1, 2])` → 1 |
| Максимум | `max(lst)` | `max([3, 1, 2])` → 3 |
| Сумма | `sum(lst)` | `sum([1, 2, 3])` → 6 |

## 5. Генераторы списков (list comprehension)

| Синтаксис | Пример |
|---|---|
| `[expr for x in seq]` | `[x*2 for x in range(5)]` |
| `[expr for x in seq if cond]` | `[x for x in range(10) if x % 2 == 0]` |
| `[expr if cond else other for x in seq]` | `["чёт" if x % 2 == 0 else "нечёт" for x in range(5)]` |

```python
squares = [x**2 for x in range(10)]
evens = [x for x in range(20) if x % 2 == 0]
pairs = [(i, j) for i in range(3) for j in range(2)]
```

## 6. Копирование списков

```python
a = [1, 2, 3]

# поверхностная копия
b = a.copy()
c = a[:]
d = list(a)

# глубокая копия (для вложенных списков)
import copy
e = copy.deepcopy([[1, 2], [3, 4]])
```

Обычное присваивание `b = a` не копирует — обе переменные ссылаются на один список.

## 7. Вложенные списки

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print(matrix[0][1])  # 2
print(matrix[1])     # [4, 5, 6]

# обход
for row in matrix:
    for item in row:
        print(item, end=" ")
```

## 8. Многомерные списки (генератором)

```python
# матрица 3x4, заполненная нулями
rows, cols = 3, 4
matrix = [[0 for _ in range(cols)] for _ in range(rows)]
```

## 9. Стек и очередь через список

### Стек (LIFO)

```python
stack = []
stack.append(1)
stack.append(2)
stack.append(3)
print(stack.pop())  # 3
print(stack.pop())  # 2
```

### Очередь (FIFO) — лучше использовать `collections.deque`

```python
from collections import deque

queue = deque(["a", "b", "c"])
queue.append("d")
print(queue.popleft())  # a
```

## 10. Распаковка списка

```python
a, b, c = [10, 20, 30]
print(a, b, c)

first, *middle, last = [1, 2, 3, 4, 5]
print(first, middle, last)
```

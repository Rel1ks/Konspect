# Работа со строками в Python

В Python **строка (`str`)** — это неизменяемая последовательность символов. Поддерживает множество методов для обработки текста. 

## 1. Создание строк

| Способ | Синтаксис | Пример |
|---|---|---|
| Одинарные кавычки | `'...'` | `'Привет'` |
| Двойные кавычки | `"..."` | `"Мир"` |
| Многострочные | `'''...'''` или `"""..."""` | `"""стр1\nстр2"""` |
| Через `str()` | `str(value)` | `str(42)` → `"42"` |

```python
s1 = 'Привет'
s2 = "Мир"
s3 = '''
Многострочная
строка
'''
```

## 2. Индексация и срезы

Каждый символ имеет индекс (с нуля). Отрицательные индексы отсчитывают с конца.

| Выражение | Описание | Пример для `s = "Python"` |
|---|---|---|
| `s[i]` | Символ по индексу | `s[0]` → `'P'` |
| `s[-1]` | Последний символ | `s[-1]` → `'n'` |
| `s[i:j]` | Срез от i до j-1 | `s[1:4]` → `'yth'` |
| `s[i:]` | От i до конца | `s[2:]` → `'thon'` |
| `s[:j]` | От начала до j-1 | `s[:4]` → `'Pyth'` |
| `s[i:j:k]` | Срез с шагом k | `s[::2]` → `'Pto'` |
| `s[::-1]` | Переворот строки | `s[::-1]` → `'nohtyP'` |

```python
s = "Python"
print(s[0])      # P
print(s[-1])     # n
print(s[1:4])    # yth
print(s[::-1])   # nohtyP
```

## 3. Основные методы строк

### Изменение регистра

| Метод | Описание | Пример |
|---|---|---|
| `s.lower()` | Нижний регистр | `"Hello".lower()` → `"hello"` |
| `s.upper()` | Верхний регистр | `"Hello".upper()` → `"HELLO"` |
| `s.capitalize()` | Первая буква заглавная | `"hello".capitalize()` → `"Hello"` |
| `s.title()` | Каждое слово с заглавной | `"hello world".title()` → `"Hello World"` |
| `s.swapcase()` | Замена регистра | `"Hello".swapcase()` → `"hELLO"` |

### Поиск и проверка

| Метод | Описание | Пример |
|---|---|---|
| `s.find(sub)` | Индекс первого вхождения (или -1) | `"abc".find("b")` → `1` |
| `s.index(sub)` | Индекс первого вхождения (ошибка если нет) | `"abc".index("b")` → `1` |
| `s.rfind(sub)` | Индекс последнего вхождения | `"aba".rfind("a")` → `2` |
| `s.count(sub)` | Количество вхождений | `"abca".count("a")` → `2` |
| `s.startswith(prefix)` | Начинается ли с prefix | `"Hello".startswith("He")` → `True` |
| `s.endswith(suffix)` | Заканчивается ли на suffix | `"Hello".endswith("lo")` → `True` |

### Проверка содержимого

| Метод | Описание | Пример |
|---|---|---|
| `s.isalpha()` | Только буквы | `"abc".isalpha()` → `True` |
| `s.isdigit()` | Только цифры | `"123".isdigit()` → `True` |
| `s.isalnum()` | Буквы и/или цифры | `"abc123".isalnum()` → `True` |
| `s.isspace()` | Только пробелы | `"   ".isspace()` → `True` |
| `s.islower()` | Все строчные | `"abc".islower()` → `True` |
| `s.isupper()` | Все заглавные | `"ABC".isupper()` → `True` |

### Пример
```python
text = "Привет, Мир!"

print(text.lower())
print(text.find("Мир"))
print(text.startswith("Привет"))
print("123".isdigit())
```

## 4. Форматирование строк

### F-строки (рекомендуемый способ)

| Синтаксис | Описание | Пример |
|---|---|---|
| `f"{var}"` | Подстановка переменной | `f"x = {x}"` |
| `f"{var:.2f}"` | Формат числа | `f"{pi:.2f}"` |
| `f"{var:>10}"` | Выравнивание вправо | `f"{name:>10}"` |
| `f"{var:<10}"` | Выравнивание влево | `f"{name:<10}"` |
| `f"{var:^10}"` | Выравнивание по центру | `f"{name:^10}"` |
| `f"{var:.3%}"` | Процентный формат | `f"{pct:.1%}"` |

```python
name = "Анна"
age = 25
pi = 3.14159

print(f"Имя: {name}, Возраст: {age}")
print(f"Число Пи: {pi:.2f}")
print(f"|{name:>10}|")
```

### `format()` и `%`

```python
# format()
print("Имя: {}, Возраст: {}".format("Анна", 25))
print("{1} {0}".format("первый", "второй"))

# % (старый стиль)
print("Имя: %s, Возраст: %d" % ("Анна", 25))
```

## 5. Разделение и объединение

| Метод | Описание | Пример |
|---|---|---|
| `s.split(sep)` | Разделить по разделителю | `"a b c".split()` → `['a', 'b', 'c']` |
| `s.rsplit(sep)` | Разделить справа | `"a-b-c".rsplit("-", 1)` |
| `s.partition(sep)` | Разделить на три части | `"a-b".partition("-")` |
| `sep.join(list)` | Объединить через разделитель | `", ".join(["a", "b"])` |
| `s.splitlines()` | Разделить по переносам | `"a\nb".splitlines()` |

```python
text = "яблоко, груша, банан"
items = text.split(", ")
print(items)

joined = " | ".join(items)
print(joined)  # яблоко | груша | банан
```

## 6. Удаление пробелов и замена

| Метод | Описание | Пример |
|---|---|---|
| `s.strip(chars)` | Удалить символы по краям | `"  text  ".strip()` → `"text"` |
| `s.lstrip(chars)` | Удалить слева | `"  text".lstrip()` → `"text"` |
| `s.rstrip(chars)` | Удалить справа | `"text  ".rstrip()` → `"text"` |
| `s.replace(old, new)` | Замена подстроки | `"a b a".replace("a", "c")` |
| `s.removeprefix(prefix)` | Удалить префикс (3.9+) | `"test.py".removeprefix("test")` |
| `s.removesuffix(suffix)` | Удалить суффикс (3.9+) | `"test.py".removesuffix(".py")` |

```python
text = "  Привет, Мир!  "
print(text.strip())

text2 = "a,b,c,a"
print(text2.replace("a", "z"))

file = "document.pdf"
print(file.removesuffix(".pdf"))
```

## 7. Длина, конкатенация и повторение

| Операция | Синтаксис | Пример |
|---|---|---|
| Длина | `len(s)` | `len("abc")` → `3` |
| Конкатенация | `+` | `"ab" + "cd"` → `"abcd"` |
| Повторение | `*` | `"ha" * 3` → `"hahaha"` |
| Содержит ли | `in` | `"a" in "cat"` → `True` |

```python
s1 = "Привет"
s2 = "Мир"

print(len(s1))
print(s1 + ", " + s2 + "!")
print("!" * 5)
print("Мир" in s1 + s2)
```

## 8. Экранированные последовательности

Символы, которые задаются обратной косой чертой.

| Последовательность | Значение |
|---|---|
| `\n` | Перевод строки |
| `\t` | Табуляция |
| `\\` | Обратная косая черта |
| `\'` | Одинарная кавычка |
| `\"` | Двойная кавычка |
| `\r` | Возврат каретки |

```python
print("Строка 1\nСтрока 2")
print("Путь: C:\\Users\\Admin")
print("Он сказал: \"Привет\"")
```

### Сырые строки (raw strings)

Позволяют игнорировать экранирование.

```python
path = r"C:\Users\Admin\Documents"
print(path)
```

## 9. Юникод и кодировки

| Функция | Описание | Пример |
|---|---|---|
| `ord(char)` | Код символа Unicode | `ord("A")` → `65` |
| `chr(code)` | Символ по коду | `chr(65)` → `"A"` |
| `s.encode(enc)` | Кодирование в байты | `"А".encode("utf-8")` |
| `s.decode(enc)` | Декодирование из байтов | `b'\xd0\x90'.decode("utf-8")` |

```python
print(ord("А"))  # 1040
print(chr(1040)) # А

text = "Привет"
encoded = text.encode("utf-8")
print(encoded)
print(encoded.decode("utf-8"))
```

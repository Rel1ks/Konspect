# Условные конструкции в Python

**Условные конструкции** позволяют выполнять разные блоки кода в зависимости от истинности выражений.

## 1. `if`, `elif`, `else`

| Конструкция | Описание |
|---|---|
| `if условие:` | Выполняется, если условие истинно |
| `elif условие:` | Проверяется, если предыдущие условия ложны |
| `else:` | Выполняется, если все условия ложны |

```python
x = int(input())

if x > 0:
    print("Положительное")
elif x == 0:
    print("Ноль")
else:
    print("Отрицательное")
```

## 2. Тернарный оператор

Компактная форма условия в одну строку.

| Синтаксис | Пример |
|---|---|
| `value if condition else other` | `"чёт" if x % 2 == 0 else "нечёт"` |

```python
age = 20
status = "взрослый" if age >= 18 else "ребёнок"
print(status)
```

## 3. Вложенные условия

Условия могут быть внутри других условий.

```python
x = 15

if x > 0:
    if x % 2 == 0:
        print("Положительное и чётное")
    else:
        print("Положительное и нечётное")
else:
    print("Не положительное")
```

## 4. Логические операторы в условиях

| Оператор | Описание | Пример |
|---|---|---|
| `and` | И (оба истинны) | `x > 0 and x < 10` |
| `or` | Или (хотя бы один истинен) | `x < 0 or x > 100` |
| `not` | Не (отрицание) | `not is_empty` |

```python
age = 25
has_ticket = True
is_vip = False

if age >= 18 and has_ticket:
    print("Проходите")

if is_vip or (age >= 18 and has_ticket):
    print("Добро пожаловать")

if not has_ticket:
    print("Купите билет")
```

## 5. Операторы сравнения

| Оператор | Описание |
|---|---|
| `==` | Равно |
| `!=` | Не равно |
| `>` | Больше |
| `<` | Меньше |
| `>=` | Больше или равно |
| `<=` | Меньше или равно |

### Цепочки сравнений

Python позволяет объединять несколько сравнений в цепочку.

```python
x = 5

print(0 < x < 10)    # True
print(1 < x < 3)     # False
print(x == 5 == 5)   # True
```

## 6. Проверка на `None`

Для проверки на `None` используют `is`, а не `==`.

```python
value = None

if value is None:
    print("Нет значения")

if value is not None:
    print("Значение есть")
```

## 7. Проверка на вхождение

```python
nums = [1, 2, 3]
name = "Анна"

if 2 in nums:
    print("2 есть в списке")

if "н" in name:
    print("Буква н есть в имени")
```

## 8. Truthy и Falsy значения

В Python любое значение можно использовать в условии. Оно будет приведено к `bool`.

| Всегда `False` | Всегда `True` |
|---|---|
| `0`, `0.0`, `0j` | Любое ненулевое число |
| `""` (пустая строка) | Любая непустая строка |
| `[]`, `()`, `{}`, `set()` | Любая непустая коллекция |
| `None` | |
| `False` | |

```python
name = input("Введите имя: ")

if name:
    print(f"Привет, {name}")
else:
    print("Имя не введено")

items = []
if not items:
    print("Список пуст")
```

## 9. `match` (Python 3.10+)

Структурное сопоставление шаблонов — аналог `switch` из других языков.

```python
status = 200

match status:
    case 200:
        print("OK")
    case 404:
        print("Не найдено")
    case 500:
        print("Ошибка сервера")
    case _:
        print("Неизвестный статус")
```

### Сопоставление с распаковкой

```python
point = (3, 5)

match point:
    case (0, 0):
        print("Начало координат")
    case (x, 0):
        print(f"На оси X: x={x}")
    case (0, y):
        print(f"На оси Y: y={y}")
    case (x, y):
        print(f"Точка: x={x}, y={y}")
```

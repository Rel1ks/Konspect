# Базовый синтаксис Python

В Python **синтаксис** — это набор правил, которые определяют, как пишутся и структурируются программы.

## 1. Комментарии

Используются для пояснения кода. Интерпретатор их игнорирует.

| Тип | Синтаксис | Пример |
|---|---|---|
| Однострочный | `#` | `# это комментарий` |
| Многострочный | `'''` или `"""` | `'''много строк'''` |

### Пример
```python
# однострочный комментарий

"""
многострочный
комментарий
"""
```

## 2. Переменные и типы данных

Переменные создаются простым присваиванием. Тип определяется автоматически.

| Тип | Описание | Пример |
|---|---|---|
| `int` | Целое число | `x = 5` |
| `float` | Число с плавающей точкой | `pi = 3.14` |
| `str` | Строка | `name = "Анна"` |
| `bool` | Логическое значение | `flag = True` |
| `NoneType` | Отсутствие значения | `data = None` |

### Пример
```python
age = 25
price = 99.9
greeting = "Привет"
is_ready = False
```

## 3. Вывод данных

Функция `print()` выводит данные в консоль.

| Синтаксис | Описание | Пример |
|---|---|---|
| `print(value)` | Вывод одного значения | `print("Hello")` |
| `print(a, b, c)` | Вывод нескольких значений | `print(1, 2, 3)` |
| `print(*values, sep=' ', end='\n')` | Разделитель и окончание | `print(1, 2, sep='-')` |

### Пример
```python
name = "Мир"
print("Привет,", name)
print(1, 2, 3, sep=", ")
print("Без переноса", end="")
```

## 4. Ввод данных

Функция `input()` читает строку с клавиатуры.

| Синтаксис | Описание | Пример |
|---|---|---|
| `input()` | Ввод строки | `name = input()` |
| `input(prompt)` | Ввод с подсказкой | `input("Имя: ")` |
| `int(input())` | Ввод числа | `age = int(input())` |

### Пример
```python
name = input("Введите имя: ")
age = int(input("Сколько лет? "))
print(f"Привет, {name}!")
```

## 5. Условные конструкции

Позволяют выполнять код в зависимости от условия.

| Конструкция | Описание | Пример |
|---|---|---|
| `if` | Если условие истинно | `if x > 0:` |
| `elif` | Иначе если | `elif x == 0:` |
| `else` | Иначе | `else:` |

### Пример
```python
x = int(input())

if x > 0:
    print("Положительное")
elif x == 0:
    print("Ноль")
else:
    print("Отрицательное")
```

## 6. Циклы

Повторяют выполнение блока кода.

| Конструкция | Описание | Пример |
|---|---|---|
| `while` | Пока условие истинно | `while x < 10:` |
| `for ... in` | Для каждого элемента | `for i in range(5):` |
| `break` | Прервать цикл | `if x == 5: break` |
| `continue` | Перейти к следующей итерации | `if x == 3: continue` |

### Пример
```python
# while
i = 0
while i < 3:
    print(i)
    i += 1

# for
for i in range(3):
    print(i)
```

## 7. Функции

Блок кода, который можно вызывать многократно.

| Синтаксис | Описание | Пример |
|---|---|---|
| `def name():` | Объявление функции | `def greet():` |
| `def name(params):` | Функция с параметрами | `def add(a, b):` |
| `return` | Возврат значения | `return a + b` |

### Пример
```python
def greet(name):
    return f"Привет, {name}!"

def add(a, b):
    return a + b

print(greet("Анна"))
print(add(3, 5))
```

## 8. Списки

Упорядоченная коллекция элементов.

| Операция | Синтаксис | Пример |
|---|---|---|
| Создание | `[ ]` | `nums = [1, 2, 3]` |
| Индексация | `list[i]` | `nums[0]` |
| Длина | `len(list)` | `len(nums)` |
| Добавление | `list.append(x)` | `nums.append(4)` |
| Срез | `list[i:j]` | `nums[1:3]` |

### Пример
```python
numbers = [10, 20, 30]
numbers.append(40)
print(numbers[0])
print(len(numbers))
print(numbers[1:3])
```

## 9. Словари

Коллекция пар «ключ — значение».

| Операция | Синтаксис | Пример |
|---|---|---|
| Создание | `{ }` | `d = {"a": 1}` |
| Доступ по ключу | `dict[key]` | `d["a"]` |
| Добавление | `dict[key] = val` | `d["b"] = 2` |
| Проверка ключа | `key in dict` | `"a" in d` |

### Пример
```python
person = {
    "name": "Анна",
    "age": 25
}

print(person["name"])
person["city"] = "Москва"
print("age" in person)
```

## 10. F-строки (форматирование)

Современный способ встраивать переменные в строки.

| Синтаксис | Описание | Пример |
|---|---|---|
| `f"текст {var}"` | Подстановка переменной | `f"Имя: {name}"` |
| `f"{var:.2f}"` | Формат числа | `f"{pi:.2f}"` |

### Пример
```python
name = "Анна"
age = 25
pi = 3.14159

print(f"Меня зовут {name}, мне {age} лет.")
print(f"Число Пи: {pi:.2f}")
```

## 11. Обработка исключений

Позволяет обрабатывать ошибки без остановки программы.

| Конструкция | Описание |
|---|---|
| `try` | Попробовать выполнить код |
| `except` | Обработать ошибку |
| `else` | Выполнить, если ошибки не было |
| `finally` | Выполнить в любом случае |

### Пример
```python
try:
    num = int(input("Введите число: "))
    print(10 / num)
except ValueError:
    print("Это не число!")
except ZeroDivisionError:
    print("Деление на ноль!")
else:
    print("Всё прошло успешно.")
finally:
    print("Блок завершён.")
```

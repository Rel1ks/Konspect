# Циклы (петли) в Python

**Циклы** позволяют многократно выполнять блок кода.

## 1. `while`

Повторяет код, пока условие истинно.

| Синтаксис | Описание |
|---|---|
| `while условие:` | Выполняется, пока условие `True` |

```python
i = 0
while i < 5:
    print(i)
    i += 1
```

### Бесконечный цикл

```python
while True:
    cmd = input("> ")
    if cmd == "exit":
        break
    print(f"Вы ввели: {cmd}")
```

## 2. `for`

Итерирует по последовательности (список, строка, range и т.д.).

| Синтаксис | Описание |
|---|---|
| `for elem in sequence:` | Для каждого элемента последовательности |

```python
# по списку
for fruit in ["яблоко", "груша", "банан"]:
    print(fruit)

# по строке
for ch in "Привет":
    print(ch)

# по словарю (ключи)
d = {"a": 1, "b": 2}
for key in d:
    print(key, d[key])
```

## 3. `range()`

Генерирует последовательность чисел. Часто используется в циклах.

| Вызов | Описание | Пример |
|---|---|---|
| `range(stop)` | 0 до stop-1 | `range(5)` → 0, 1, 2, 3, 4 |
| `range(start, stop)` | start до stop-1 | `range(2, 5)` → 2, 3, 4 |
| `range(start, stop, step)` | с шагом step | `range(0, 10, 2)` → 0, 2, 4, 6, 8 |

```python
for i in range(5):
    print(i)

for i in range(2, 10, 3):
    print(i)
```

## 4. `break` и `continue`

| Инструкция | Описание |
|---|---|
| `break` | Немедленно прерывает цикл |
| `continue` | Переходит к следующей итерации |

```python
# break — выход из цикла
for i in range(10):
    if i == 5:
        break
    print(i)  # 0, 1, 2, 3, 4

# continue — пропуск итерации
for i in range(5):
    if i == 2:
        continue
    print(i)  # 0, 1, 3, 4
```

## 5. `else` в циклах

Блок `else` выполняется, если цикл завершился **не через `break`**.

```python
# else сработает
for i in range(5):
    print(i)
else:
    print("Цикл завершён")  # выполнится

# else НЕ сработает
for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("Не выполнится из-за break")
```

## 6. `enumerate()`

Возвращает пару (индекс, элемент) при итерации.

```python
fruits = ["яблоко", "груша", "банан"]

for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# с начальным индексом
for i, fruit in enumerate(fruits, start=1):
    print(f"{i}. {fruit}")
```

## 7. `zip()`

Объединяет несколько последовательностей в кортежи.

```python
names = ["Анна", "Борис", "Вика"]
ages = [25, 30, 22]

for name, age in zip(names, ages):
    print(f"{name}: {age} лет")
```

## 8. `reversed()` и `sorted()`

Обход последовательности в обратном порядке или отсортированной.

```python
nums = [3, 1, 4, 1, 5]

for n in reversed(nums):
    print(n)

for n in sorted(nums):
    print(n)
```

## 9. Вложенные циклы

Цикл внутри другого цикла.

```python
for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")
```

### Генерация таблицы умножения

```python
for i in range(1, 10):
    for j in range(1, 10):
        print(f"{i}*{j}={i*j:2}", end="  ")
    print()
```

## 10. Генераторы списков (list comprehension)

Компактный способ создания списков через цикл.

| Синтаксис | Описание | Пример |
|---|---|---|
| `[expr for x in seq]` | Простой | `[x*2 for x in range(5)]` |
| `[expr for x in seq if cond]` | С фильтром | `[x for x in range(10) if x % 2 == 0]` |

```python
squares = [x**2 for x in range(10)]
print(squares)

evens = [x for x in range(20) if x % 2 == 0]
print(evens)

pairs = [(i, j) for i in range(3) for j in range(2)]
print(pairs)
```

## 11. Dict и set comprehension

```python
# dict comprehension
squares_dict = {x: x**2 for x in range(5)}
print(squares_dict)

# set comprehension
unique_len = {len(word) for word in ["я", "ты", "он", "она"]}
print(unique_len)
```

# Приведение типов (type casting) в Python

**Приведение типов** — преобразование значения из одного типа данных в другой.

## 1. Явное приведение (explicit)

Выполняется вручную с помощью встроенных функций.

### В числа

| Функция | Из чего | Описание | Пример |
|---|---|---|---|
| `int(x)` | `str`, `float` | Целое число | `int(3.9)` → `3` |
| `float(x)` | `str`, `int` | Число с плавающей точкой | `float("3.14")` → `3.14` |
| `complex(x)` | `int`, `float` | Комплексное число | `complex(5)` → `(5+0j)` |

```python
print(int(3.99))        # 3 (дробная часть отбрасывается)
print(int("42"))        # 42
print(float("3.14"))    # 3.14
print(float(10))        # 10.0
print(complex(3))       # (3+0j)
```

### В строку

| Функция | Из чего | Пример |
|---|---|---|
| `str(x)` | Любой тип | `str(42)` → `"42"` |

```python
print(str(123))         # "123"
print(str(3.14))        # "3.14"
print(str([1, 2, 3]))   # "[1, 2, 3]"
print(str(True))        # "True"
```

### В булево значение

| Функция | Из чего | Пример |
|---|---|---|
| `bool(x)` | Любой тип | `bool(1)` → `True` |

```python
print(bool(1))          # True
print(bool(0))          # False
print(bool(""))         # False
print(bool("text"))     # True
print(bool([]))         # False
print(bool([1, 2]))     # True
```

### В коллекции

| Функция | Из чего | Пример |
|---|---|---|
| `list(x)` | Итерируемый объект | `list("abc")` → `["a", "b", "c"]` |
| `tuple(x)` | Итерируемый объект | `tuple([1, 2])` → `(1, 2)` |
| `set(x)` | Итерируемый объект | `set([1, 2, 2])` → `{1, 2}` |
| `frozenset(x)` | Итерируемый объект | `frozenset([1, 2])` |
| `dict(x)` | Последовательность пар | `dict([("a", 1)])` → `{"a": 1}` |

```python
print(list("hello"))          # ["h", "e", "l", "l", "o"]
print(tuple([1, 2, 3]))       # (1, 2, 3)
print(set([1, 2, 2, 3]))      # {1, 2, 3} (дубликаты удалены)
print(dict([("a", 1), ("b", 2)]))  # {"a": 1, "b": 2}
```

## 2. Неявное приведение (implicit)

Python автоматически приводит типы, когда это безопасно.

```python
# int + float → float
print(5 + 3.14)   # 8.14 (int 5 стал float)

# int + complex → complex
print(5 + 2j)     # (5+2j)

# bool + int → int
print(True + 2)   # 3 (True = 1)
print(False + 5)  # 5 (False = 0)
```

### Когда неявное приведение НЕ работает

```python
# Ошибка — нельзя сложить str и int
# print(" age: " + 25)  # TypeError

# Нужно явное приведение
print(" age: " + str(25))
```

## 3. Безопасное приведение

### Проверка перед преобразованием

```python
value = "123"

if value.isdigit():
    num = int(value)
    print(num)
else:
    print("Невозможно преобразовать")
```

### try/exept для обработки ошибок

```python
def safe_int(value, default=0):
    try:
        return int(value)
    except (ValueError, TypeError):
        return default

print(safe_int("123"))     # 123
print(safe_int("abc"))     # 0
print(safe_int(None))      # 0
```

## 4. Особенности `int()` из `float`

Преобразование `float` → `int` всегда **отбрасывает дробную часть** (не округляет).

```python
print(int(3.999))    # 3
print(int(-3.999))   # -3
print(int(5.0))      # 5
```

### Альтернативы — округление

```python
import math

print(math.floor(3.9))    # 3 (вниз)
print(math.ceil(3.1))     # 4 (вверх)
print(round(3.5))          # 4 (математическое округление)
print(round(3.4))          # 3
```

## 5. Приведение в коллекции

### Распаковка в отдельные переменные

```python
# str → list (по символам)
chars = list("Привет")
print(chars)  # ["П", "р", "и", "в", "е", "т"]

# list → str (через join)
words = ["Hello", "World"]
print(" ".join(words))  # "Hello World"
```

### Изменяемые и неизменяемые коллекции

```python
# list → tuple (неизменяемый)
lst = [1, 2, 3]
tpl = tuple(lst)

# tuple → list (изменяемый)
tpl = (1, 2, 3)
lst = list(tpl)

# list → set (удаление дубликатов)
nums = [1, 2, 2, 3, 3, 3]
unique = list(set(nums))
print(unique)  # [1, 2, 3]
```

## 6. ASCII и Unicode коды

```python
# символ → код
print(ord("A"))    # 65
print(ord("А"))    # 1040

# код → символ
print(chr(65))     # A
print(chr(1040))   # А

# строка → байты
text = "Привет"
bytes_data = text.encode("utf-8")
print(bytes_data)   # b'\xd0\x9f\xd1\x80...'

# байты → строка
decoded = bytes_data.decode("utf-8")
print(decoded)      # "Привет"
```

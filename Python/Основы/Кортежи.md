# Кортежи (tuple) в Python

**Кортеж** — неизменяемая упорядоченная коллекция элементов. Часто используется для хранения разнородных данных фиксированной структуры.

## 1. Создание

| Способ | Пример |
|---|---|
| Круглые скобки | `(1, 2, 3)` |
| Без скобок | `1, 2, 3` |
| Один элемент (с запятой) | `(1,)` |
| `tuple()` | `tuple("abc")` → `("a", "b", "c")` |

```python
empty = ()
nums = (1, 2, 3)
without_parens = 10, 20, 30
single = (5,)  # без запятой будет int
chars = tuple("hello")
```

## 2. Индексация и срезы

Работают так же, как у списков.

```python
t = (10, 20, 30, 40, 50)

print(t[0])      # 10
print(t[-1])     # 50
print(t[1:4])    # (20, 30, 40)
print(t[::-1])   # (50, 40, 30, 20, 10)
```

## 3. Основные операции

| Операция | Синтаксис | Пример |
|---|---|---|
| Длина | `len(t)` | `len((1, 2, 3))` → 3 |
| Конкатенация | `+` | `(1, 2) + (3, 4)` → `(1, 2, 3, 4)` |
| Повторение | `*` | `(0,) * 3` → `(0, 0, 0)` |
| Проверка | `in` | `2 in (1, 2, 3)` → True |
| Поиск | `t.index(x)` | `(1, 2, 3).index(2)` → 1 |
| Подсчёт | `t.count(x)` | `(1, 2, 2, 3).count(2)` → 2 |
| Минимум | `min(t)` | `min((3, 1, 2))` → 1 |
| Максимум | `max(t)` | `max((3, 1, 2))` → 3 |
| Сумма | `sum(t)` | `sum((1, 2, 3))` → 6 |

```python
a = (1, 2, 3)
b = (4, 5)
print(a + b)
print(a * 2)
print(2 in a)
```

## 4. Неизменяемость

Кортеж нельзя изменить после создания.

```python
t = (1, 2, 3)
# t[0] = 10  # TypeError

# Но если внутри изменяемый объект — его можно изменить
t = (1, [2, 3], 4)
t[1].append(4)
print(t)  # (1, [2, 3, 4], 4)
```

## 5. Распаковка кортежа

```python
point = (3, 5)
x, y = point
print(f"x={x}, y={y}")

# с помощью *
a, b, *rest = (1, 2, 3, 4, 5)
print(a, b, rest)  # 1 2 [3, 4, 5]

# обмен переменных
a, b = 1, 2
a, b = b, a
print(a, b)  # 2 1
```

## 6. Кортежи как ключи словаря

В отличие от списков, кортежи можно использовать как ключи.

```python
locations = {
    (55.75, 37.62): "Москва",
    (59.93, 30.31): "СПб"
}

print(locations[(55.75, 37.62)])
```

## 7. Возврат нескольких значений из функции

```python
def divide(a, b):
    quotient = a // b
    remainder = a % b
    return quotient, remainder

q, r = divide(10, 3)
print(q, r)  # 3 1
```

## 8. Когда использовать кортеж вместо списка

| Кортеж | Список |
|---|---|
| Неизменяемый | Изменяемый |
| Быстрее | Медленнее |
| Можно как ключ dict | Нельзя как ключ dict |
| Для фиксированных данных | Для изменяемых данных |
| Гарантия неизменности | Нет гарантии |

## 9. Методы кортежа

У кортежей всего два метода:

```python
t = (1, 2, 2, 3, 2)

print(t.count(2))  # 3
print(t.index(3))  # 3
```

## 10. Сравнение кортежей

Сравниваются поэлементно слева направо.

```python
print((1, 2, 3) < (1, 2, 4))    # True
print((1, 2) < (1, 2, 3))       # True
print(("a", "b") == ("a", "b")) # True
print((1, "a") < (2, "b"))      # True
```

# Операторы в Python

В Python **операторы** — это специальные символы и ключевые слова, с помощью которых выполняют вычисления, сравнения, присваивание и проверку условий. 

## 1. Арифметические операторы

Используются для математических действий.

| Оператор | Значение | Пример | Результат |
|---|---|---:|---:|
| `+` | Сложение | `2 + 3` | `5` |
| `-` | Вычитание | `5 - 2` | `3` |
| `*` | Умножение | `4 * 3` | `12` |
| `/` | Деление | `10 / 2` | `5.0` |
| `//` | Целочисленное деление | `10 // 3` | `3` |
| `%` | Остаток от деления | `10 % 3` | `1` |
| `**` | Возведение в степень | `2 ** 3` | `8` |

### Пример
```python
a = 10
b = 3

print(a + b)
print(a // b)
print(a % b)
print(a ** b)
```

## 2. Операторы сравнения

Сравнивают значения и возвращают `True` или `False`.

| Оператор | Значение | Пример |
|---|---|---|
| `==` | Равно | `a == b` |
| `!=` | Не равно | `a != b` |
| `>` | Больше | `a > b` |
| `<` | Меньше | `a < b` |
| `>=` | Больше или равно | `a >= b` |
| `<=` | Меньше или равно | `a <= b` |

### Пример
```python
x = 7
y = 10

print(x < y)
print(x == y)
```

## 3. Операторы присваивания

Нужны для записи значения в переменную.

| Оператор | Значение | Пример |
|---|---|---|
| `=` | Присваивание | `a = 5` |
| `+=` | Прибавить и присвоить | `a += 2` |
| `-=` | Вычесть и присвоить | `a -= 2` |
| `*=` | Умножить и присвоить | `a *= 2` |
| `/=` | Разделить и присвоить | `a /= 2` |
| `//=` | Целочисленно разделить и присвоить | `a //= 2` |
| `%=` | Остаток и присвоить | `a %= 2` |
| `**=` | Возвести в степень и присвоить | `a **= 2` |

### Пример
```python
a = 5
a += 3
print(a)  # 8
```

## 4. Логические операторы

Используются в условиях.

| Оператор | Значение | Пример |
|---|---|---|
| `and` | И | `x > 0 and x < 10` |
| `or` | Или | `x < 0 or x > 100` |
| `not` | Не | `not True` |

### Пример
```python
age = 20
has_ticket = True

print(age >= 18 and has_ticket)
```

## 5. Операторы принадлежности

Проверяют, есть ли элемент в последовательности.

| Оператор | Значение | Пример |
|---|---|---|
| `in` | Есть в | `'a' in 'cat'` |
| `not in` | Нет в | `'z' not in 'cat'` |

### Пример
```python
nums = [1, 2, 3, 4]

print(3 in nums)
print(5 not in nums)
```

## 6. Операторы тождественности

Проверяют, это один и тот же объект или нет.

| Оператор | Значение | Пример |
|---|---|---|
| `is` | Один и тот же объект | `a is b` |
| `is not` | Разные объекты | `a is not b` |

### Пример
```python
a = [1, 2]
b = a
c = [1, 2]

print(a is b)
print(a is c)
```

## 7. Побитовые операторы

Работают с числами на уровне битов.

| Оператор | Значение | Пример |
|---|---|---|
| `&` | Побитовое И | `5 & 3` |
| `\|` | Побитовое ИЛИ | `5 \| 3` |
| `^` | Побитовое XOR | `5 ^ 3` |
| `~` | Побитовое НЕ | `~5` |
| `<<` | Сдвиг влево | `5 << 1` |
| `>>` | Сдвиг вправо | `5 >> 1` |

### Пример
```python
a = 5
b = 3

print(a & b)
print(a | b)
print(a << 1)
```

## 8. Порядок выполнения

Если в выражении несколько операторов, Python выполняет их по приоритету:

1. Скобки `()`
2. Возведение в степень `**`
3. Умножение, деление `* / // %`
4. Сложение и вычитание `+ -`
5. Сравнения
6. `not`
7. `and`
8. `or`

### Пример
```python
result = 2 + 3 * 4
print(result)  # 14
```

# Функции в Python

**Функция** — именованный блок кода, который принимает аргументы, выполняет действия и (опционально) возвращает результат.

## 1. Определение и вызов

| Синтаксис | Описание |
|---|---|
| `def name():` | Функция без параметров |
| `def name(params):` | Функция с параметрами |
| `return value` | Возврат значения |

```python
def greet():
    print("Привет!")

def add(a, b):
    return a + b

greet()
result = add(3, 5)
print(result)
```

## 2. Параметры и аргументы

### Позиционные аргументы

Передаются в том же порядке, что и параметры.

```python
def describe(name, age, city):
    print(f"{name}, {age} лет, {city}")

describe("Анна", 25, "Москва")
```

### Именованные аргументы (keyword args)

```python
describe(name="Борис", city="СПб", age=30)
```

### Значения по умолчанию

```python
def greet(name, greeting="Привет"):
    print(f"{greeting}, {name}!")

greet("Анна")
greet("Борис", "Здравствуй")
```

### `*args` — произвольное количество позиционных аргументов

```python
def sum_all(*args):
    return sum(args)

print(sum_all(1, 2, 3))
print(sum_all(10, 20))
```

### `**kwargs` — произвольное количество именованных аргументов

```python
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Анна", age=25, city="Москва")
```

### Комбинирование

Порядок: `обычные` → `*args` → `**kwargs`

```python
def func(a, b, *args, option=True, **kwargs):
    print(a, b, args, option, kwargs)

func(1, 2, 3, 4, option=False, x=10, y=20)
```

## 3. `return`

Функция может возвращать одно или несколько значений.

```python
def min_max(lst):
    return min(lst), max(lst)

low, high = min_max([3, 1, 7, 2, 9])
print(low, high)
```

Если `return` не указан — функция возвращает `None`.

## 4. Типизация (type hints)

Подсказки типов для читаемости и проверки кода.

```python
def add(a: int, b: int) -> int:
    return a + b

def greet(name: str) -> str:
    return f"Привет, {name}!"

def process(items: list[int]) -> list[int]:
    return [x * 2 for x in items]
```

## 5. Области видимости (scope)

| Область | Описание |
|---|---|
| `local` | Внутри функции |
| `enclosing` | Внешняя функция (для замыканий) |
| `global` | Глобальная (модуль) |
| `built-in` | Встроенные имена Python |

### global и nonlocal

```python
counter = 0

def increment():
    global counter
    counter += 1

increment()
print(counter)  # 1
```

```python
def outer():
    x = 10

    def inner():
        nonlocal x
        x += 5
        return x

    return inner

f = outer()
print(f())  # 15
```

## 6. lambda (анонимные функции)

Однострочная функция без имени.

| Синтаксис | Пример |
|---|---|
| `lambda args: expression` | `lambda x: x ** 2` |

```python
square = lambda x: x ** 2
print(square(5))

# обычно используется с map/filter/sorted
nums = [1, 2, 3, 4, 5]
squares = list(map(lambda x: x ** 2, nums))
evens = list(filter(lambda x: x % 2 == 0, nums))
print(squares)
print(evens)
```

## 7. Функции как объекты первого класса

Функции можно передавать, присваивать, возвращать.

```python
def apply(func, value):
    return func(value)

def double(x):
    return x * 2

print(apply(double, 5))

# функции в списке
operations = [double, lambda x: x ** 2, lambda x: x + 1]
for op in operations:
    print(op(3))
```

## 8. Декораторы

Функция, которая оборачивает другую функцию.

```python
def timer(func):
    def wrapper(*args, **kwargs):
        import time
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"{func.__name__}: {end - start:.4f} сек")
        return result
    return wrapper

@timer
def slow_sum(n):
    return sum(range(n))

print(slow_sum(10_000_000))
```

## 9. Замыкания (closures)

Внутренняя функция запоминает переменные внешней.

```python
def make_multiplier(n):
    def multiplier(x):
        return x * n
    return multiplier

double = make_multiplier(2)
triple = make_multiplier(3)

print(double(5))
print(triple(5))
```

## 10. Рекурсия

Функция вызывает саму себя.

```python
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)

def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

print(factorial(5))   # 120
print(fibonacci(10))  # 55
```

## 11. Docstrings

Документация прямо в теле функции.

```python
def add(a, b):
    """
    Складывает два числа.

    Args:
        a: первое число
        b: второе число

    Returns:
        Сумма a и b
    """
    return a + b

# просмотр документации
print(add.__doc__)
# help(add)
```

# Встроенные функции (Built-in)

Python предоставляет множество встроенных функций — для их вызова не нужно ничего импортировать.

## 1. Работа с типами

| Функция | Описание | Пример |
|---|---|---|
| `int(x)` | В целое число | `int("42")` |
| `float(x)` | В число с точкой | `float("3.14")` |
| `str(x)` | В строку | `str(100)` |
| `bool(x)` | В булево | `bool(0)` |
| `list(x)` | В список | `list("abc")` |
| `tuple(x)` | В кортеж | `tuple([1, 2])` |
| `set(x)` | Во множество | `set([1, 2, 2])` |
| `dict(x)` | В словарь | `dict([("a", 1)])` |
| `type(x)` | Тип объекта | `type(5)` → `int` |

## 2. Ввод/вывод

| Функция | Описание | Пример |
|---|---|---|
| `print(...)` | Вывод в консоль | `print("Hello")` |
| `input(prompt)` | Ввод строки | `input("> ")` |
| `format(value, spec)` | Форматирование | `format(3.14, ".2f")` |

## 3. Работа с последовательностями

| Функция | Описание | Пример |
|---|---|---|
| `len(x)` | Длина | `len([1, 2, 3])` → 3 |
| `range(start, stop, step)` | Диапазон чисел | `range(5)` |
| `enumerate(seq)` | Индекс + элемент | `enumerate(["a", "b"])` |
| `zip(*iterables)` | Объединение | `zip([1, 2], ["a", "b"])` |
| `map(func, seq)` | Применить функцию | `map(str, [1, 2])` |
| `filter(func, seq)` | Отфильтровать | `filter(None, [0, 1, 0])` |
| `reversed(seq)` | В обратном порядке | `reversed([1, 2, 3])` |
| `sorted(seq, key, reverse)` | Сортировка | `sorted([3, 1, 2])` |
| `slice(start, stop, step)` | Срез | `slice(1, 3)` |

```python
print(list(map(lambda x: x * 2, [1, 2, 3])))
print(list(filter(lambda x: x > 0, [-1, 0, 2])))
print(sorted([3, 1, 4, 1, 5]))
```

## 4. Математические

| Функция | Описание | Пример |
|---|---|---|
| `abs(x)` | Модуль числа | `abs(-5)` → 5 |
| `sum(iterable)` | Сумма | `sum([1, 2, 3])` → 6 |
| `min(iterable)` | Минимум | `min(3, 1, 2)` → 1 |
| `max(iterable)` | Максимум | `max(3, 1, 2)` → 3 |
| `pow(x, y)` | Степень (как `**`) | `pow(2, 3)` → 8 |
| `round(x, ndigits)` | Округление | `round(3.1415, 2)` → 3.14 |
| `divmod(a, b)` | (a // b, a % b) | `divmod(10, 3)` → (3, 1) |

## 5. Проверки и логика

| Функция | Описание | Пример |
|---|---|---|
| `all(iterable)` | Все элементы истинны | `all([True, 1, "a"])` |
| `any(iterable)` | Хотя бы один истинен | `any([False, 0, "a"])` |
| `isinstance(obj, type)` | Проверка типа | `isinstance(5, int)` |
| `issubclass(cls, base)` | Проверка наследования | `issubclass(bool, int)` |
| `callable(obj)` | Можно ли вызвать | `callable(len)` → True |

```python
print(all([1, 2, 3]))     # True
print(any([0, "", None])) # False
```

## 6. Итераторы и генераторы

| Функция | Описание | Пример |
|---|---|---|
| `iter(obj)` | Получить итератор | `iter([1, 2])` |
| `next(iterator)` | Следующий элемент | `next(it)` |
| `id(obj)` | Уникальный ID объекта | `id(x)` |
| `hash(obj)` | Хеш объекта | `hash("text")` |

## 7. Атрибуты и интроспекция

| Функция | Описание |
|---|---|
| `dir(obj)` | Список атрибутов объекта |
| `getattr(obj, name)` | Получить атрибут |
| `setattr(obj, name, value)` | Установить атрибут |
| `hasattr(obj, name)` | Проверить наличие атрибута |
| `help(obj)` | Документация |

```python
print(dir([]))
print(getattr("hello", "upper")())
print(hasattr([], "append"))
```

## 8. Прочие полезные

| Функция | Описание | Пример |
|---|---|---|
| `open(file, mode)` | Открыть файл | `open("f.txt", "r")` |
| `repr(obj)` | Строковое представление для разработчика | `repr("a")` → `"'a'"` |
| `eval(expr)` | Выполнить выражение | `eval("2 + 3")` |
| `exec(code)` | Выполнить код | `exec("x = 5")` |
| `globals()` | Глобальные переменные | `globals()` |
| `locals()` | Локальные переменные | `locals()` |
| `vars(obj)` | Атрибуты объекта | `vars()` |
| `input(prompt)` | Ввод строки | `input("> ")` |
| `print(...)` | Вывод | `print(1, 2, 3)` |

```python
print(repr("hello"))
print(eval("2 * 3 + 1"))
```

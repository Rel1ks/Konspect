# Исключения в Python

**Исключения** — это ошибки, возникающие во время выполнения программы. Python позволяет их перехватывать и обрабатывать.

## 1. Базовый `try`/`except`

| Конструкция | Описание |
|---|---|
| `try:` | Код, который может вызвать ошибку |
| `except:` | Выполняется при возникновении ошибки |

```python
try:
    num = int(input("Введите число: "))
    print(10 / num)
except:
    print("Произошла ошибка")
```

## 2. Обработка конкретных исключений

Лучше перехватывать конкретные типы, а не все подряд.

| Тип исключения | Когда возникает |
|---|---|
| `ValueError` | Неверное значение (например, `int("abc")`) |
| `ZeroDivisionError` | Деление на ноль |
| `TypeError` | Операция над несовместимым типом |
| `IndexError` | Обращение за границы списка |
| `KeyError` | Отсутствующий ключ в словаре |
| `FileNotFoundError` | Файл не найден |
| `AttributeError` | Несуществующий атрибут |
| `ImportError` | Ошибка импорта модуля |

```python
try:
    num = int(input())
    print(10 / num)
except ValueError:
    print("Это не число")
except ZeroDivisionError:
    print("Деление на ноль")
```

## 3. `else` и `finally`

| Блок | Описание |
|---|---|
| `else:` | Выполняется, если не было исключения |
| `finally:` | Выполняется **всегда** (даже если было исключение или `return`) |

```python
try:
    f = open("file.txt", "r")
    content = f.read()
except FileNotFoundError:
    print("Файл не найден")
else:
    print("Файл прочитан успешно")
    print(content)
finally:
    print("Блок завершён")
    f.close()
```

## 4. Получение информации об ошибке

```python
try:
    x = 1 / 0
except ZeroDivisionError as e:
    print(f"Тип: {type(e).__name__}")
    print(f"Сообщение: {e}")
```

## 5. `raise` — выброс исключения

Можно вручную вызывать исключения.

```python
def set_age(age):
    if age < 0:
        raise ValueError("Возраст не может быть отрицательным")
    if age > 150:
        raise ValueError("Некорректный возраст")
    print(f"Возраст: {age}")

try:
    set_age(-5)
except ValueError as e:
    print(e)
```

### Проброс исключения

```python
try:
    x = int("abc")
except ValueError:
    print("Логирование ошибки")
    raise  # пробрасываем дальше
```

## 6. Создание своих исключений

```python
class CustomError(Exception):
    pass

class NegativeNumberError(ValueError):
    def __init__(self, value):
        self.value = value
        super().__init__(f"Отрицательное число: {value}")

try:
    raise NegativeNumberError(-10)
except NegativeNumberError as e:
    print(e)
```

## 7. `assert` — проверка утверждений

Если условие ложно — выбрасывается `AssertionError`.

```python
def divide(a, b):
    assert b != 0, "Делитель не может быть нулём"
    return a / b

try:
    divide(10, 0)
except AssertionError as e:
    print(e)
```

```python
# except Exception перехватит почти все ошибки
try:
    risky_code()
except Exception as e:
    print(f"Поймано: {type(e).__name__}: {e}")
```

## 9. Групповые исключения (Python 3.11+)

```python
try:
    x = int("abc")
except (ValueError, TypeError, ZeroDivisionError) as e:
    print(f"Ошибка: {e}")
```

## 10. Блокировка ошибок `suppress` (Python 3.11+)

```python
from contextlib import suppress

with suppress(FileNotFoundError):
    # ошибка будет проигнорирована
    open("nope.txt")
```

## 11. Практические примеры

### Повтор запроса при ошибке ввода

```python
while True:
    try:
        age = int(input("Сколько лет? "))
        if age < 0:
            raise ValueError("Возраст не может быть отрицательным")
        break
    except ValueError as e:
        print(f"Ошибка: {e}. Попробуйте снова.")

print(f"Ваш возраст: {age}")
```

### Безопасное чтение файла

```python
def read_file(path):
    try:
        with open(path, "r", encoding="utf-8") as f:
            return f.read()
    except FileNotFoundError:
        return "Файл не найден"
    except PermissionError:
        return "Нет доступа к файлу"
    except Exception as e:
        return f"Неизвестная ошибка: {e}"
```

### Контекстный менеджер с обработкой

```python
try:
    with open("data.txt", "r") as f:
        data = f.read()
except FileNotFoundError:
    data = "{}"

print(data)
```

# Переменные и типы данных в Python

В Python **переменные** — это именованные ссылки на объекты в памяти. Тип данных определяется автоматически во время выполнения (динамическая типизация). 

## 1. Присваивание переменных

Переменная создаётся в момент первого присваивания.

| Синтаксис | Описание | Пример |
|---|---|---|
| `name = value` | Простое присваивание | `x = 10` |
| `a, b = v1, v2` | Множественное присваивание | `x, y = 1, 2` |
| `a = b = value` | Цепочное присваивание | `a = b = 0` |
| `_` | Переменная-заглушка | `_, b = (1, 2)` |

### Пример
```python
x = 5
name, age = "Анна", 25
a = b = c = 0
```

## 2. Основные встроенные типы

Все данные в Python — объекты. Вот базовые типы:

### Числовые типы

| Тип | Описание | Пример | Функция |
|---|---|---|---|
| `int` | Целое число (без ограничения) | `x = 42` | `int()` |
| `float` | Число с плавающей точкой | `pi = 3.14` | `float()` |
| `complex` | Комплексное число | `c = 1 + 2j` | `complex()` |

```python
a = 10
b = 3.14
c = 1 + 2j

print(type(a))  # <class 'int'>
print(type(b))  # <class 'float'>
print(type(c))  # <class 'complex'>
```

### Строки (`str`)

Неизменяемая последовательность символов.

| Синтаксис | Описание | Пример |
|---|---|---|
| `'текст'` | Одинарные кавычки | `'Привет'` |
| `"текст"` | Двойные кавычки | `"Привет"` |
| `'''текст'''` | Многострочная строка | `'''стр1\nстр2'''` |
| `f"текст {var}"` | F-строка (интерполяция) | `f"x = {x}"` |

```python
s1 = 'Привет'
s2 = "Мир"
s3 = '''
Много
строк
'''
```

### Булев тип (`bool`)

Принимает только два значения.

| Значение | Описание |
|---|---|
| `True` | Истина (численно = 1) |
| `False` | Ложь (численно = 0) |

```python
is_active = True
is_admin = False
print(int(True))   # 1
print(int(False))  # 0
```

### None (`NoneType`)

Обозначает отсутствие значения.

```python
data = None
result = print("test")  # None — print ничего не возвращает
```

## 3. Функция `type()`

Позволяет узнать тип объекта.

| Синтаксис | Описание | Пример |
|---|---|---|
| `type(obj)` | Возвращает тип | `type(5)` → `int` |
| `isinstance(obj, type)` | Проверка типа | `isinstance(5, int)` → `True` |

### Пример
```python
x = 3.14
print(type(x))                # <class 'float'>
print(isinstance(x, float))   # True
```

## 4. Преобразование типов

Явное приведение одного типа к другому.

| Функция | Из чего | Во что | Пример |
|---|---|---|---|
| `int(x)` | str, float | `int` | `int("42")` → `42` |
| `float(x)` | str, int | `float` | `float("3.14")` → `3.14` |
| `str(x)` | любой тип | `str` | `str(100)` → `"100"` |
| `bool(x)` | любой тип | `bool` | `bool(0)` → `False` |

```python
# str -> int
num = int("42")

# float -> int (с отбрасыванием дробной части)
print(int(3.99))  # 3

# int -> str
text = str(100)
```

## 5. Правила именования переменных

| Правило | Пример ✅ | Пример ❌ |
|---|---|---|
| Только буквы, цифры, `_` | `my_var` | `my-var` |
| Не начинается с цифры | `var1` | `1var` |
| Регистр важен | `name != Name` | — |
| Не совпадает с ключевыми словами | `value` | `class`, `for` |

### Ключевые слова (зарезервированы)

```
False, None, True, and, as, assert, async, await, break,
class, continue, def, del, elif, else, except, finally,
for, from, global, if, import, in, is, lambda, nonlocal,
not, or, pass, raise, return, try, while, with, yield
```

### Стили именования

| Стиль | Пример | Где используется |
|---|---|---|
| `snake_case` | `my_variable` | переменные, функции |
| `UPPER_CASE` | `MAX_SIZE` | константы |
| `_private` | `_internal` | внутренние (по соглашению) |
| `__dunder__` | `__init__` | магические методы |

## 6. Динамическая типизация

Одна и та же переменная может хранить значения разных типов.

```python
x = 10       # int
x = "текст"  # str — теперь это строка
x = [1, 2]   # list — теперь это список
```

## 7. Изменяемые и неизменяемые типы

| Категория | Типы | Описание |
|---|---|---|
| **Изменяемые** (mutable) | `list`, `dict`, `set`, `bytearray` | Можно изменить содержимое |
| **Неизменяемые** (immutable) | `int`, `float`, `str`, `bool`, `tuple`, `frozenset`, `bytes` | Нельзя изменить после создания |

```python
# неизменяемый — str
s = "hello"
# s[0] = "H"  # Ошибка!

# изменяемый — list
lst = [1, 2, 3]
lst[0] = 100  # Ok
```

## 8. Проверка типа в условиях

В Python многие значения приводятся к `bool` автоматически.

| Значение | `bool(x)` |
|---|---|
| `0`, `0.0`, `""`, `[]`, `{}`, `None` | `False` |
| Любое другое | `True` |

```python
name = input("Имя: ")
if name:  # True, если строка не пуста
    print(f"Привет, {name}!")
else:
    print("Имя не введено")
```

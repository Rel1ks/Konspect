```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n

        a, b = 1, 2
        for _ in range(3, n + 1):
            a, b = b, a + b
        return b
```

- Задача: найти количество способов подняться на `n` ступенек, шагая на 1 или 2 ступени.
- Если `n = 1` или `n = 2`, ответ равен `n`.
- Количество способов для каждой ступеньки равно сумме способов для двух предыдущих
- Используется идея последовательности Фибоначчи.
- Хранятся только два последних значения, чтобы не тратить лишнюю память.
- Цикл последовательно считает ответ до ступеньки `n`.
- В конце возвращается общее количество способов.

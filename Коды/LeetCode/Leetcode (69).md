```typescript
function mySqrt(x: number): number {
    if (x < 2) return x;
    
    let left = 0;
    let right = x;
    
    while (left <= right) {
        const mid = Math.floor((left + right) / 2);
        const square = mid * mid;
        
        if (square === x) {
            return mid;
        } else if (square < x) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    
    return right;
}


```

Функция `mySqrt` принимает целое неотрицательное число `x` и возвращает **целый квадратный корень** из `x` (то есть `⌊√x⌋`, без дробной части).

1. **Обработка простых случаев**  
Если x < 2, функция сразу возвращает x, потому что:
- √0 = 0
- √1 = 1

2. **Инициализация границ поиска**  
Переменные:

- `left = 0` — левая граница диапазона
- `right = x` — правая граница диапазона

Квадратный корень числа x всегда лежит в этом диапазоне.

3. **Бинарный поиск**  
Пока `left <= right`:
- Вычисляется середина диапазона mid
- Считается квадрат mid * mid        
Далее:
- Если `mid² === x`, найден точный квадратный корень — возвращаем `mid`
- Если `mid² < x`, значит корень больше — сдвигаем `left` вправо
- Если `mid² > x`, значит корень меньше — сдвигаем `right` влево

4. **Возврат результата**  
Когда цикл заканчивается, точного квадратного корня нет.  
В этом случае `right` содержит **наибольшее целое число**, квадрат которого **меньше либо равен `x`**, поэтому возвращается `right`.

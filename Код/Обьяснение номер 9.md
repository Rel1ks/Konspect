1. Проверка на отрицательные числа  
   ```python
   if x < 0:
       return False
   ```
Любое отрицательное число сразу считается не палиндромом, потому что знак - нарушает симметрию.  

2. Преобразование числа в строку  
   ```python
   str(x)
   ```
Число переводится в строку, чтобы можно было легко сравнить символы.

3. Переворот строки  
   ```python
   str(x)[::-1]
   ```
   [::-1] создаёт новую строку, где символы идут в обратном порядке.  

4. Сравнение  
   ```python
   return str(x) == str(x)[::-1]
   ```
   - Если строка равна своей перевёрнутой версии → число палиндром.  
   - Если нет → возвращается False.

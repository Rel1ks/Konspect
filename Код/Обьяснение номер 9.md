1.Преобразование числа в список символов:

  

 ```python

    z = []

    for i in str(x):

        z.append(i)

  ```

  

    Это создает список z, где каждый элемент — это символ из строки, представляющей число x.

  

2.Проверка на отрицательное число:

  
  

```python

    if z[0] == "-":

        return False

```

  

    Если первым символом  "-" , функция сразу возвращает False.

  

3.Четная длинна строки

  
  

```python

        if len(z) % 2 == 0:

            return False

```

  
  

Если длина строки чётная — вернуть False

  
  

4.Сравнение первого и последнего символа.

  
  

```python

        else:

            if z[0] == z[-1]:

                return True

            else:

                return False

```

Если длина нечётная — сравниваем первый и последний символ. Если равны — возвращаем True, иначе False.

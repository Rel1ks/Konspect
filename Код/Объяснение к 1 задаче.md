1.  Внешний цикл

```python

for i in range(len(nums)):

```

Перебираем все элементы массива по индексу i

range(len(nums)) создает последовательность от 0 до (длина_массива - 1)

  

2. Внутренний цикл

```python

for j in range(i + 1, len(nums)):

```

Для каждого i перебираем все последующие элементы по индексу j

i + 1 - начинаем со следующего элемента после текущего i

Это гарантирует, что мы не используем один и тот же элемент дважды

  

3. Проверка суммы

```python

if nums[i] + nums[j] == target:

    return [i, j]

```

Проверяем, равна ли сумма текущих элементов целевому значению

Если да - возвращаем индексы [i, j]

#БазаДанных #Таблица
База данных (БД) - это совокупность данных организованных по определённым правилам, отражающая состояние объектов и их отношений в некоторой предметной области. Предназначенная для хранения постоянной памяти и её применения. Базу данных можно рассматривать как информационную модель в предметной области.
Предметная область - это целевая тема конкретного проекта. В свою очередь доменная концептуальная модель призвана выразить значение терминов и понятий, а так же установление верных взаимосвязей между различными понятиями. 
### 3 основных типа БД
1. Реляционные БД - ещё называют табличными, что данные в них можно представить в виде таблиц.

 ![[Pasted image 20260217095847.png]]

2. Сетевые БД - в сетевых базах  данных может быть несколько различных взаимосвязей между таблицами, в том числе несколько предков. Особенность сетевой БД в том что в ней запоминаются все связи для каждой из них. Иначе говоря поиск в такой базе данных индексируется.

![[Pasted image 20260217095936.png]]


3. Иерархические БД - это модель организующая данные в древовидную структуру, где каждый дочерний элемент подчиняется только 1 родительскому

### Вывод
Вывод: каждый тип баз данных (БД) имеет свою область применения, где его преимущества максимальны, а недостатки минимальны. Универсальных решений не существует — лучшие архитектуры часто объединяют несколько типов баз данных, обеспечивая оптимальную производительность для различных частей системы.

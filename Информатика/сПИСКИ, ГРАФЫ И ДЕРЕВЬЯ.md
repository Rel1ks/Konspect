### Ведение

Структура данных это программная единица данных позволяющие хранить и обрабатывать однотипные или логически связанные данные. Для добавления, поиска, изменения и удаления данных (взаимодействия) с данными структура данных предоставляет некоторый набор функций составляющий ее интерфейс.

### сВЯЗАНЫЙ СПИСОК

Это базовый динамическая структура данных состоящая из узлов каждый из которых содержит значение и ссылку на следующий узел. Двойной связанный список это список котором узлы содержат ссылки еще и  на предыдущий элемент.

### Деревья

Это структура данных расширяющая идею связанного списка. Однако в отличии от них узлы могут более одного связанного с ними узла. Узлы дерева любое количество дочерних узлов благодаря чему данные представлены в виде гибкой схемы ветвления.

### Графы

При помощи графов можно расширить образ применения деревьев и ослабить строгую родительскую связь в них. узлы графа не имеют явной иерархии, каждый из узлов может быть соединен с любым другим узлом.

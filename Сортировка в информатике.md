# Сортировка в информатике

**Сортировка** — процесс упорядочивания элементов массива (списка) по возрастанию или убыванию согласно определённому правилу.

---

## Классификация сортировок

| Критерий | Виды |
|----------|-----|
| **По устойчивости** | Устойчивые, неустойчивые |
| **По использованию памяти** | Внутренние (в ОЗУ), внешние (с диском) |
| **По методу работы** | Сравнением, поразрядные |

**Устойчивость** — сохранение относительного порядка элементов с одинаковыми ключами после сортировки.

---

## Основные алгоритмы сортировки

### 1. Сортировка пузырьком (Bubble Sort)

**Сложность:** O(n²)  
**Устойчивость:** устойчивая  
**Память:** O(1)

Многократно меняет местами соседние элементы, если они находятся в неправильном порядке.

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
```

---

### 2. Сортировка выбором (Selection Sort)

**Сложность:** O(n²)  
**Устойчивость:** неустойчивая  
**Память:** O(1)

Находит минимальный элемент в неотсортированной части и ставит его в начало.

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_idx = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
```

---

### 3. Сортировка вставками (Insertion Sort)

**Сложность:** O(n²)  
**Устойчивость:** устойчивая  
**Память:** O(1)

Элемент вставляется в отсортированную часть на своё место.

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
```

---

### 4. Быстрая сортировка (Quick Sort)

**Сложность:** O(n log n) средняя, O(n²) худшая  
**Устойчивость:** неустойчивая  
**Память:** O(log n)

Алгоритм «разделяй и властвуй»: выбирается опорный элемент, массив делится на части.

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

---

### 5. Сортировка слиянием (Merge Sort)

**Сложность:** O(n log n)  
**Устойчивость:** устойчивая  
**Память:** O(n)

Массив делится пополам, затем части сливаются в отсортированный массив.

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result
```

### 6. Пирамидальная сортировка (Heap Sort)

**Сложность:** O(n log n)  
**Устойчивость:** неустойчивая  
**Память:** O(1)

Использует структуру данных «куча» (пирамида).

```python
def heap_sort(arr):
    n = len(arr)
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    for i in range(n - 1, 0, -1):
        arr[0], arr[i] = arr[i], arr[0]
        heapify(arr, i, 0)

def heapify(arr, n, i):
    largest = i
    l = 2 * i + 1
    r = 2 * i + 2
    if l < n and arr[l] > arr[largest]:
        largest = l
    if r < n and arr[r] > arr[largest]:
        largest = r
    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)
```

### 7. Сортировка подсчётом (Counting Sort)

**Сложность:** O(n + k), где k — диапазон значений  
**Устойчивость:** устойчивая  
**Память:** O(k)

Подсчитывает количество вхождений каждого значения.

```python
def counting_sort(arr):
    if not arr:
        return arr
    max_val = max(arr)
    count = [0] * (max_val + 1)
    for num in arr:
        count[num] += 1
    result = []
    for i, c in enumerate(count):
        result.extend([i] * c)
    return result
```

### 8. Поразрядная сортировка (Radix Sort)

**Сложность:** O(n · k), где k — количество разрядов  
**Устойчивость:** устойчивая  
**Память:** O(n + k)

Сортирует числа поразрядно (от младшего к старшему или наоборот).

```python
def radix_sort(arr):
    if not arr:
        return arr
    max_val = max(arr)
    exp = 1
    while max_val // exp > 0:
        counting_sort_by_digit(arr, exp)
        exp *= 10
    return arr

def counting_sort_by_digit(arr, exp):
    n = len(arr)
    output = [0] * n
    count = [0] * 10
    for i in range(n):
        index = (arr[i] // exp) % 10
        count[index] += 1
    for i in range(1, 10):
        count[i] += count[i - 1]
    for i in range(n - 1, -1, -1):
        index = (arr[i] // exp) % 10
        output[count[index] - 1] = arr[i]
        count[index] -= 1
    for i in range(n):
        arr[i] = output[i]
```

### 9. Сортировка Шелла (Shell Sort)

**Сложность:** O(n log n) — O(n²) в зависимости от последовательности  
**Устойчивость:** неустойчивая  
**Память:** O(1)

Улучшенная версия сортировки вставками с использованием интервалов.

```python
def shell_sort(arr):
    n = len(arr)
    gap = n // 2
    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i
            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap
            arr[j] = temp
        gap //= 2
```

---

### 10. Сортировка кучей (Tim Sort)

**Сложность:** O(n log n)  
**Устойчивость:** устойчивая  
**Память:** O(n)

Гибрид сортировки вставками и слиянием. Используется в Python (`sorted()`, `list.sort()`).

## Когда какую сортировку использовать

- **Маленькие массивы (n < 50)** — вставками или пузырьком
- **Требование устойчивости** — слиянием, Tim Sort, подсчётом
- **Ограниченная память** — быстрая, пирамидальная, Шелла
- **Известный диапазон значений** — подсчётом или поразрядная
- **Универсальный выбор** — быстрая или слиянием
